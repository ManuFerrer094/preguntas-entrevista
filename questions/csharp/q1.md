En C#, un **delegado** es un tipo que representa una referencia a un método con una firma determinada. Los **eventos** son una capa de encapsulación sobre los delegados que implementan el patrón publicador-suscriptor.

**Delegados:**

```csharp
// Declarar un delegado
public delegate void Notificar(string mensaje);

// Usarlo
Notificar notificacion = (msg) => Console.WriteLine($"Mensaje: {msg}");
notificacion("Hola mundo");
```

Los delegados predefinidos `Action`, `Func` y `Predicate` evitan declarar tipos propios en la mayoría de los casos:

```csharp
Action<string> mostrar = msg => Console.WriteLine(msg);
Func<int, int, int> sumar = (a, b) => a + b;
Predicate<int> esPositivo = n => n > 0;
```

**Eventos — ejemplo con botón (patrón estándar):**

```csharp
public class Boton
{
    // 1. Declarar el evento usando EventHandler
    public event EventHandler<string> Click;

    // 2. Método protegido que dispara el evento
    protected virtual void OnClick(string texto)
        => Click?.Invoke(this, texto);

    public void Pulsar(string texto) => OnClick(texto);
}

public class Formulario
{
    public Formulario()
    {
        var boton = new Boton();

        // 3. Suscribirse al evento
        boton.Click += Boton_Click;
        boton.Pulsar("Guardar");

        // 4. Desuscribirse cuando ya no se necesite
        boton.Click -= Boton_Click;
    }

    private void Boton_Click(object sender, string texto)
        => Console.WriteLine($"Boton pulsado: {texto}");
}
```

**Evento con datos personalizados (EventArgs):**

```csharp
public class DatosCambiadosEventArgs : EventArgs
{
    public string ValorAnterior { get; }
    public string ValorNuevo { get; }

    public DatosCambiadosEventArgs(string anterior, string nuevo)
    {
        ValorAnterior = anterior;
        ValorNuevo = nuevo;
    }
}

public class Editor
{
    public event EventHandler<DatosCambiadosEventArgs> DatosCambiados;

    private string _texto = string.Empty;

    public string Texto
    {
        get => _texto;
        set
        {
            var anterior = _texto;
            _texto = value;
            DatosCambiados?.Invoke(this, new DatosCambiadosEventArgs(anterior, value));
        }
    }
}
```

**Puntos clave:**
- Usa `event` para exponer delegados: evita que terceros invoquen o reemplacen el delegado directamente.
- Llama al evento con `?.Invoke(...)` para prevenir `NullReferenceException` cuando no hay suscriptores.
- Desuscríbete con `-=` para evitar fugas de memoria, especialmente en controles de larga vida.
