Java es un lenguaje de programación de alto nivel, orientado a objetos y multiplataforma, diseñado para tener la menor cantidad de dependencias de implementación posible. Creado en 1995, es ampliamente utilizado para desarrollar aplicaciones empresariales, aplicaciones móviles (Android), software web y sistemas embebidos, destacando por su seguridad, robustez y portabilidad bajo el lema "escribir una vez, ejecutar en cualquier lugar" (WORA)
