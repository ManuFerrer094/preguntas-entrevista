---
title: ¿Cuál es la diferencia entre JDK, JRE y JVM?
difficulty: easy
tags: [jdk]
author: "Manuel Ferrer"
authorUrl: "https://www.linkedin.com/in/manuferrer/"
---

El JDK (Java Development Kit) es para desarrollo, JRE (Java Runtime Environment) para ejecutar, y la JVM (Java Virtual Machine) es la máquina que ejecuta el bytecode.
