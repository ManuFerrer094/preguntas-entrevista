# ¿Qué es el Testing unitario?

El Testing unitario prueba unidades individuales de código de forma aislada.

---

# ¿Qué es TDD?

TDD (Test Driven Development) es una metodología donde primero se escriben los tests y luego el código.

Ciclo: Red → Green → Refactor.

---

# ¿Qué es un Mock en Testing?

Un Mock es un objeto simulado que imita el comportamiento de objetos reales de forma controlada.

---

# ¿Qué es Cypress?

Cypress es un framework de testing end-to-end para aplicaciones web modernas que se ejecuta directamente en el navegador.

---

# ¿Qué es la pirámide de testing?

La pirámide de testing describe la proporción ideal: muchos unit tests, algunos integration tests, pocos E2E tests.
