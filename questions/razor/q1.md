---
title: ¿Qué es Razor y cómo se diferencia de otras plantillas de servidor como ASPX?
difficulty: easy
tags: [razor, aspnet, plantillas, webforms]
---

**Razor** es un motor de plantillas (view engine) introducido en ASP.NET MVC 3 que permite mezclar código C# (o VB.NET) con HTML mediante una sintaxis ligera basada en el símbolo `@`. Está diseñado para ser limpio, minimalista y fácil de leer.

**Diferencias principales con ASPX (Web Forms):**

| Característica | Razor | ASPX |
|---|---|---|
| Sintaxis | `@` para transiciones código/HTML | `<% %>` y `<%= %>` |
| Legibilidad | Alta, menos ruido sintáctico | Menor, más verbosa |
| Detección de bloques | Automática por el parser | Manual con etiquetas |
| IntelliSense | Completo en Visual Studio | Completo en Visual Studio |
| Uso recomendado | ASP.NET MVC / Razor Pages | Legacy Web Forms |

**Ejemplo comparativo:**

```razor
@* Razor *@
<ul>
@foreach (var item in Model.Items)
{
    <li>@item.Name</li>
}
</ul>
```

```aspx
<!-- ASPX -->
<ul>
<% foreach (var item in Model.Items) { %>
    <li><%= item.Name %></li>
<% } %>
</ul>
```

Razor es el estándar moderno en el ecosistema .NET para renderizado de vistas en el lado del servidor.
