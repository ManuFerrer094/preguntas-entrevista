---
title: ¿Cómo se implementa la carga de archivos (file upload) en Razor con IFormFile?
difficulty: medium
tags: [razor, file-upload, iformfile, iformfilecollection, aspnet-core]
---

ASP.NET Core gestiona la carga de archivos mediante la interfaz `IFormFile`, que proporciona acceso a los datos del archivo subido incluyendo nombre, tipo MIME, tamaño y un stream de lectura.

**Vista Razor:**

```razor
<form asp-action="Subir" method="post" enctype="multipart/form-data">
    <input type="file" asp-for="Archivo" accept=".pdf,.jpg,.png" />
    <button type="submit">Subir</button>
    <span asp-validation-for="Archivo" class="text-danger"></span>
</form>
```

> Es obligatorio `enctype="multipart/form-data"` para que el navegador envíe los archivos.

**ViewModel:**

```csharp
public class SubirArchivoViewModel
{
    [Required]
    [DataType(DataType.Upload)]
    public IFormFile Archivo { get; set; }
}
```

**Controlador / PageModel:**

```csharp
[HttpPost]
public async Task<IActionResult> Subir(SubirArchivoViewModel model)
{
    if (!ModelState.IsValid) return View(model);

    var archivo = model.Archivo;

    // Validar tamaño y extensión
    if (archivo.Length > 5 * 1024 * 1024)
    {
        ModelState.AddModelError("Archivo", "El archivo no puede superar 5 MB.");
        return View(model);
    }

    var extensionesPermitidas = new[] { ".pdf", ".jpg", ".png" };
    var extension = Path.GetExtension(archivo.FileName).ToLowerInvariant();
    if (!extensionesPermitidas.Contains(extension))
    {
        ModelState.AddModelError("Archivo", "Tipo de archivo no permitido.");
        return View(model);
    }

    // Guardar con nombre seguro (nunca usar FileName del cliente directamente)
    var nombreSeguro = Path.GetRandomFileName() + extension;
    var ruta = Path.Combine("wwwroot/uploads", nombreSeguro);

    using var stream = System.IO.File.Create(ruta);
    await archivo.CopyToAsync(stream);

    return RedirectToAction("Exito");
}
```

**Múltiples archivos:** usa `List<IFormFile>` o `IFormFileCollection`.
