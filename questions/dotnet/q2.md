---
title: ¿Qué servicios proporciona el Common Language Runtime (CLR)?
difficulty: easy
tags: [dotnet, clr, memoria, cts]
---

El **Common Language Runtime (CLR)** es el corazón del .NET Framework y provee los siguientes servicios esenciales:

- **Administración de memoria:** A través del **Garbage Collector (GC)**, el CLR libera automáticamente los objetos que ya no están en uso, evitando fugas de memoria sin intervención del desarrollador.

- **Common Type System (CTS):** Define cómo se declaran y usan los tipos en el runtime, garantizando la interoperabilidad entre diferentes lenguajes .NET (C#, VB.NET, F#, etc.). Todos los lenguajes comparten el mismo sistema de tipos.

- **Compilación Just-In-Time (JIT):** El código fuente se compila primero a código intermedio (CIL - Common Intermediate Language) y luego el JIT lo convierte a código máquina nativo en tiempo de ejecución.

- **Manejo de excepciones:** Proporciona un sistema de manejo de excepciones estructurado y consistente entre lenguajes.

- **Seguridad:** Incluye seguridad de acceso al código (Code Access Security) y verificación de tipos para evitar operaciones no seguras.

- **Interoperabilidad:** Permite la comunicación con código no administrado (COM, APIs nativas de Windows) mediante P/Invoke y COM Interop.
