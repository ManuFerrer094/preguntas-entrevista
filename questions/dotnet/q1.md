---
title: ¿Qué es el .NET Framework y cuáles son sus componentes principales?
difficulty: easy
tags: [dotnet, clr, framework, arquitectura]
---

El **.NET Framework** es una plataforma de desarrollo de Microsoft que proporciona un entorno de ejecución administrado y una extensa biblioteca de clases para construir y ejecutar aplicaciones Windows.

Sus dos componentes principales son:

- **Common Language Runtime (CLR):** Es el motor de ejecución del framework. Se encarga de la administración de memoria (recolección de basura), la seguridad, el manejo de excepciones, la compilación del código intermedio (CIL/MSIL) a código nativo, y la interoperabilidad entre lenguajes mediante el Common Type System (CTS).

- **Biblioteca de clases base (BCL / Framework Class Library):** Conjunto de tipos y APIs reutilizables que cubren funcionalidades como acceso a datos, entrada/salida de archivos, seguridad, redes, interfaces de usuario (WinForms, WPF) y mucho más.

.NET 4.8 es la última versión del .NET Framework clásico, destinado a aplicaciones Windows. A diferencia de .NET Core / .NET 5+, está diseñado exclusivamente para Windows y mantiene soporte a largo plazo como parte del sistema operativo.
