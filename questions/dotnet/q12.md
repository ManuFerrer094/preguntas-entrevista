---
title: ¿Qué es Minimal API en .NET 6+ y cuándo es preferible a los controladores MVC?
difficulty: medium
tags: [dotnet, minimal-api, aspnet-core, rest, endpoints]
---

**Minimal APIs** es un estilo de construcción de APIs HTTP introducido en .NET 6 que elimina el código repetitivo de los controladores MVC, permitiendo definir endpoints directamente en `Program.cs` o en clases auxiliares con muy pocas líneas.

**Ejemplo completo en `Program.cs`:**

```csharp
var builder = WebApplication.CreateBuilder(args);
builder.Services.AddDbContext<AppDbContext>(opt =>
    opt.UseInMemoryDatabase("Productos"));

var app = builder.Build();

// GET all
app.MapGet("/productos", async (AppDbContext db) =>
    await db.Productos.ToListAsync());

// GET by id
app.MapGet("/productos/{id}", async (int id, AppDbContext db) =>
    await db.Productos.FindAsync(id) is Producto p
        ? Results.Ok(p)
        : Results.NotFound());

// POST
app.MapPost("/productos", async (Producto producto, AppDbContext db) =>
{
    db.Productos.Add(producto);
    await db.SaveChangesAsync();
    return Results.Created($"/productos/{producto.Id}", producto);
});

// PUT
app.MapPut("/productos/{id}", async (int id, Producto input, AppDbContext db) =>
{
    var p = await db.Productos.FindAsync(id);
    if (p is null) return Results.NotFound();
    p.Nombre = input.Nombre;
    p.Precio = input.Precio;
    await db.SaveChangesAsync();
    return Results.NoContent();
});

// DELETE
app.MapDelete("/productos/{id}", async (int id, AppDbContext db) =>
{
    if (await db.Productos.FindAsync(id) is Producto p)
    {
        db.Productos.Remove(p);
        await db.SaveChangesAsync();
        return Results.NoContent();
    }
    return Results.NotFound();
});

app.Run();
```

**Cuándo usar Minimal APIs vs Controladores MVC:**

| Criterio | Minimal API | Controladores MVC |
|---|---|---|
| Proyectos pequeños/microservicios | ✅ Ideal | Excesivo |
| Necesidad de filtros, model binding complejo | Limitado | ✅ Completo |
| OpenAPI/Swagger | Soportado | Soportado |
| Organización para proyectos grandes | Puede complicarse | ✅ Mejor estructura |

A partir de .NET 7+, Minimal APIs admiten grupos de rutas (`MapGroup`) y validación de filtros para acercar su funcionalidad a los controladores.
