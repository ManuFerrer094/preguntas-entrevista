Un array es una estructura ordenada que permite almacenar varios valores bajo una misma variable. En JavaScript, los arrays son objetos especializados y pueden contener valores de tipos distintos, aunque en buen diseño normalmente buscas colecciones homogéneas.

## Ejemplo de código
```js
const notas = [8, 9, 6, 10];
const mezcla = ['Ana', 42, true, { activo: true }];

console.log(Array.isArray(notas)); // true
console.log(notas.length); // 4
```

## Idea clave
Un array sirve cuando el orden importa y cuando quieres aplicar operaciones de colección: iterar, filtrar, transformar, agrupar o buscar.

## Matices importantes
- Aunque uses índices numéricos, un array sigue siendo un objeto.
- JavaScript no te garantiza la idea clásica de "memoria contigua" a nivel del lenguaje como explicación pedagógica principal.
- Si lo que necesitas es una estructura clave-valor por nombre, probablemente quieres un objeto o un `Map`, no un array.

## Para interiorizarlo
No pienses en un array como "una bolsa de cosas". Piensa en él como una colección ordenada sobre la que quieres aplicar operaciones repetibles.
