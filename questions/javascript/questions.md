# ¿Qué es el Hoisting en JavaScript?

El Hoisting es el comportamiento de JavaScript de mover las declaraciones al inicio del scope durante la fase de compilación.

---

# ¿Qué es el Closure en JavaScript?

Un Closure es una función que recuerda y tiene acceso a las variables de su scope léxico incluso cuando la función se ejecuta fuera de ese scope.

---

# ¿Qué es Promise y async/await?

Las Promises representan un valor que puede estar disponible ahora, en el futuro, o nunca. `async/await` es azúcar sintáctico sobre las Promises.

---

# ¿Qué es el Event Loop en JavaScript?

El Event Loop es el mecanismo que permite a JavaScript ejecutar código asíncrono siendo single-threaded.

---

# ¿Qué es el Prototype en JavaScript?

El prototipo es el mecanismo mediante el cual los objetos JavaScript heredan características de otros objetos.
