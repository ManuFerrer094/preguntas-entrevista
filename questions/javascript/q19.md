Hay varias formas de recorrer un array y no todas sirven para lo mismo. Elegir bien la herramienta mejora claridad y evita errores.

## Ejemplo de código
```js
const numeros = [1, 2, 3];

for (const numero of numeros) {
  console.log(numero);
}

numeros.forEach((numero) => {
  console.log(numero * 2);
});

const cuadrados = numeros.map((numero) => numero ** 2);
console.log(cuadrados); // [1, 4, 9]
```

## Cuándo usar cada una
- `for` o `for...of`: cuando necesitas control fino o romper el bucle.
- `forEach`: cuando solo quieres ejecutar un efecto por elemento.
- `map`: cuando quieres transformar y devolver un nuevo array.
- `filter`: cuando quieres quedarte solo con algunos elementos.

## Para interiorizarlo
No recorras arrays por inercia. Pregunta primero qué intención tienes: leer, transformar, filtrar, buscar o acumular.
