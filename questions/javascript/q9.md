Puedes concatenar strings con el operador `+` o con template literals, usando comillas invertidas. En código moderno, las template literals suelen ser más legibles porque permiten interpolación, multilínea y menos ruido visual.

## Ejemplo de código
```js
const nombre = 'Marta';
const edad = 31;

const conMas = 'Hola, ' + nombre + '. Tienes ' + edad + ' años.';
const conTemplate = `Hola, ${nombre}. Tienes ${edad} años.`;

console.log(conMas);
console.log(conTemplate);
```

## Matices importantes
- `+` funciona bien para casos simples.
- Las template literals son mejores cuando mezclas varias variables, expresiones o saltos de línea.
- Si concatenas dentro de bucles grandes o construyes mucho HTML, a veces conviene replantear el enfoque en lugar de solo cambiar sintaxis.

## Para interiorizarlo
El aprendizaje importante aquí no es la sintaxis, sino la legibilidad. Un código correcto pero difícil de leer también te ralentiza.
