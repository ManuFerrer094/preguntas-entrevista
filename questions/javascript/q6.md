Los operadores aritméticos básicos de JavaScript son `+`, `-`, `*`, `/`, `%` y `**` para potencia. También existen `++` y `--` para incrementar y decrementar.

## Ejemplo de código
```js
const a = 10;
const b = 3;

console.log(a + b); // 13
console.log(a - b); // 7
console.log(a * b); // 30
console.log(a / b); // 3.333...
console.log(a % b); // 1
console.log(a ** b); // 1000
```

## Matices importantes
- `/` siempre devuelve un número en coma flotante; JavaScript no separa enteros y decimales dentro del tipo `number`.
- `%` devuelve el resto, que en negativos puede sorprender si vienes de otros lenguajes.
- El operador `+` también concatena strings, así que conviene saber en qué contexto estás.

## Para interiorizarlo
Dominar los operadores no es memorizar símbolos, sino entender qué tipo de dato estás combinando y qué resultado produce esa combinación.
