title: ¿Qué hace el operador typeof en JavaScript?

`typeof` devuelve una cadena que describe el tipo del operando. Es útil para inspecciones rápidas, validaciones básicas y debugging.

## Ejemplo de código
```js
console.log(typeof 'hola'); // 'string'
console.log(typeof 42); // 'number'
console.log(typeof true); // 'boolean'
console.log(typeof undefined); // 'undefined'
console.log(typeof Symbol('id')); // 'symbol'
console.log(typeof 10n); // 'bigint'
console.log(typeof {}); // 'object'
console.log(typeof []); // 'object'
console.log(typeof null); // 'object'
```

## Matices importantes
- `typeof null` devuelve `'object'`, un comportamiento histórico del lenguaje.
- Para arrays es mejor `Array.isArray(valor)`.
- Para distinguir objetos complejos a veces necesitas comprobaciones más específicas que `typeof`.

## Para interiorizarlo
`typeof` es una herramienta útil, pero no mágica. Aprende cuándo basta y cuándo necesitas una validación más rica del dato.
