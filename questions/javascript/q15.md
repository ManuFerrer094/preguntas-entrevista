Las sentencias condicionales permiten ejecutar distintos bloques de código según se cumpla o no una condición. Las más habituales son `if`, `else if`, `else`, además de `switch` y el operador ternario para casos simples.

## Ejemplo de código
```js
const nota = 7;

if (nota >= 9) {
  console.log('sobresaliente');
} else if (nota >= 5) {
  console.log('aprobado');
} else {
  console.log('suspenso');
}
```

## Idea clave
Lo importante no es memorizar la forma, sino escribir condiciones claras, expresivas y con pocos caminos ambiguos.

## Matices importantes
- Una condición no necesita ser un booleano puro; JavaScript aplica truthiness/falsiness.
- Si la lógica crece demasiado, quizá ya no necesitas más `if`, sino otra estructura: un mapa, una tabla de decisiones o una función separada.
- El operador ternario va bien para expresiones simples, no para árboles complejos.

## Para interiorizarlo
Cuando escribas un condicional, pregúntate si expresa una regla de negocio clara o si solo está ocultando complejidad desordenada.
