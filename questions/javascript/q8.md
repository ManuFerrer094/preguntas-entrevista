Los operadores lógicos principales son `&&` (AND), `||` (OR) y `!` (NOT). Sirven para combinar condiciones, pero en JavaScript tienen además un detalle muy importante: devuelven operandos reales, no siempre un booleano puro.

## Ejemplo de código
```js
const usuario = 'Ana';
const fallback = 'Invitado';

console.log(true && 'ok'); // 'ok'
console.log(false && 'ok'); // false
console.log(usuario || fallback); // 'Ana'
console.log(!usuario); // false
```

## Idea clave
Estos operadores hacen short-circuit:
- `&&` deja de evaluar cuando encuentra algo falsy.
- `||` deja de evaluar cuando encuentra algo truthy.

Eso los hace útiles para validaciones, valores por defecto y ejecución condicional.

## Matices importantes
- `0`, `''`, `null`, `undefined` y `NaN` son falsy.
- A veces conviene `??` en lugar de `||` si solo quieres tratar `null` y `undefined` como ausencia.

## Para interiorizarlo
No memorices solo que `&&` es AND y `||` es OR. Aprende que forman parte del flujo de evaluación del lenguaje.
