Los elementos de un array se acceden normalmente por índice, empezando en `0`. También puedes usar métodos modernos como `.at()` cuando quieres una lectura más expresiva, por ejemplo para acceder al último elemento.

## Ejemplo de código
```js
const colores = ['rojo', 'verde', 'azul'];

console.log(colores[0]); // 'rojo'
console.log(colores[2]); // 'azul'
console.log(colores.at(-1)); // 'azul'
console.log(colores[99]); // undefined
```

## Matices importantes
- Si accedes a una posición que no existe, obtienes `undefined`, no un error automático.
- `.at(-1)` es muy útil para leer desde el final sin hacer `array[array.length - 1]`.
- Que un índice exista no significa que el valor sea siempre válido; podría ser `undefined` de verdad porque alguien lo guardó así.

## Para interiorizarlo
Cada vez que accedas a un array, piensa si das por hecho que esa posición existe. Muchas excepciones en producción vienen de asumir índices válidos sin comprobar datos reales.
