Una función es un bloque reutilizable de código que puede recibir entradas, ejecutar lógica y devolver un resultado. En JavaScript, además, las funciones son ciudadanos de primera clase: puedes guardarlas en variables, pasarlas como argumento o devolverlas desde otra función.

## Ejemplo de código
```js
function sumar(a, b) {
  return a + b;
}

const operacion = sumar;
console.log(operacion(2, 3)); // 5
```

## Idea clave
Una función no es solo "algo que se llama". En JavaScript también es un valor. Esa idea abre la puerta a callbacks, closures, composición y programación funcional.

## Matices importantes
- Una función crea su propio contexto de ejecución cuando se invoca.
- Puede o no devolver un valor explícito.
- También puede actuar como método, constructor o callback según cómo la uses.

## Para interiorizarlo
Cuando estudies funciones, no pienses solo en sintaxis. Piensa en responsabilidades: qué entra, qué sale y qué efectos secundarios produce.
