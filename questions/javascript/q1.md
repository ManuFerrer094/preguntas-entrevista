JavaScript es un lenguaje de programación de alto nivel, dinámico y multiparadigma. Nació para añadir comportamiento a las páginas web, pero hoy también se usa en servidores, herramientas de build, aplicaciones de escritorio y automatización. La especificación que define el lenguaje se llama ECMAScript, y distintos motores como V8, SpiderMonkey o JavaScriptCore se encargan de ejecutarlo.

## Idea clave
Aprender JavaScript de verdad implica entender dos cosas a la vez:
- el lenguaje en sí mismo: tipos, funciones, objetos, scope, asincronía;
- y el entorno donde corre: navegador, Node.js, workers, APIs del DOM, etc.

JavaScript no es "solo para el navegador" ni tampoco es "Java con otra sintaxis". Tiene identidad propia: funciones de primera clase, objetos prototipales, tipado dinámico y un modelo asíncrono muy potente.

## Ejemplo de código
```js
const usuario = 'Lucía';

function saludar(nombre) {
  return `Hola, ${nombre}`;
}

console.log(saludar(usuario));
```

Aquí ya aparecen varias piezas fundamentales del lenguaje: variables, strings, funciones y llamada a una API del entorno (`console.log`).

## Matices importantes
- JavaScript suele compilarse justo a tiempo internamente; no es correcto reducirlo a "lenguaje interpretado" a secas.
- El lenguaje es single-threaded en su ejecución principal, pero puede coordinar trabajo asíncrono y usar workers.
- El DOM, `fetch` o `localStorage` no son JavaScript puro: son APIs del entorno.

## Para interiorizarlo
Piensa en JavaScript como el lenguaje y en el navegador o Node.js como la "máquina" que le da superpoderes adicionales.
