Los operadores de comparación permiten construir expresiones que devuelven `true` o `false`. Los más usados son `<`, `<=`, `>`, `>=`, `===`, `!==` y, con más cuidado, `==` y `!=`.

## Ejemplo de código
```js
const edad = 20;

console.log(edad >= 18); // true
console.log(edad < 65); // true
console.log(edad === 20); // true
console.log(edad !== '20'); // true
```

## Matices importantes
- Comparar strings usa orden lexicográfico, no "sentido numérico".
- Comparar objetos no compara su contenido, sino si son exactamente la misma referencia.
- Con `==` y `!=` puede haber coerción de tipos.

## Ejemplo con objetos
```js
const a = { valor: 1 };
const b = { valor: 1 };
const c = a;

console.log(a === b); // false
console.log(a === c); // true
```

## Para interiorizarlo
Cuando una comparación te sorprenda, comprueba dos cosas: el tipo real de cada operando y si estás comparando valores o referencias.
