---
title: ¿Cómo funciona Array.prototype.flat() y flatMap() en JavaScript?
difficulty: easy
tags: [javascript, array, flat, flatMap, es2019]
---

`Array.flat()` aplana arrays anidados a una profundidad especificada. `Array.flatMap()` combina un `map()` seguido de un `flat(1)` en una sola operación, siendo más eficiente que hacerlos por separado.

## Array.flat() — Aplanar arrays anidados

```javascript
// Anidado a 1 nivel (por defecto)
const anidado = [1, [2, 3], [4, [5, 6]]];
console.log(anidado.flat());        // [1, 2, 3, 4, [5, 6]]

// Anidado a 2 niveles
console.log(anidado.flat(2));       // [1, 2, 3, 4, 5, 6]

// Profundidad infinita: Infinity
const muyAnidado = [1, [2, [3, [4, [5]]]]];
console.log(muyAnidado.flat(Infinity));  // [1, 2, 3, 4, 5]

// Elimina huecos (sparse arrays)
const conHuecos = [1, , 3, , 5];  // sparse array
console.log(conHuecos.flat());    // [1, 3, 5] — elimina huecos

// Caso práctico: estructura de árbol plana
const categorias = [
  { nombre: 'Electrónica', productos: ['Laptop', 'Tablet'] },
  { nombre: 'Libros', productos: ['Novela', 'Técnico', 'Comics'] },
  { nombre: 'Ropa', productos: ['Camiseta', 'Pantalón'] }
];

const todosLosProductos = categorias.map(c => c.productos).flat();
// ['Laptop', 'Tablet', 'Novela', 'Técnico', 'Comics', 'Camiseta', 'Pantalón']

// Con flatMap (más eficiente):
const todosLosProductosFM = categorias.flatMap(c => c.productos);
// Mismo resultado, una sola pasada
```

## Array.flatMap() — Map + Flat en uno

```javascript
// flatMap = map() seguido de flat(1)
// Es más eficiente que .map().flat() porque hace solo una pasada

// Ejemplo básico
const numeros = [1, 2, 3, 4];

// Con map + flat (dos pasadas):
const resultado = numeros.map(n => [n, n * 2]).flat();
// [1, 2, 2, 4, 3, 6, 4, 8]

// Con flatMap (una pasada, más eficiente):
const resultadoFM = numeros.flatMap(n => [n, n * 2]);
// [1, 2, 2, 4, 3, 6, 4, 8]

// Caso: expandir cada elemento en varios
const frases = ['Hola mundo', 'JavaScript es genial'];
const palabras = frases.flatMap(frase => frase.split(' '));
// ['Hola', 'mundo', 'JavaScript', 'es', 'genial']
```

## Casos de uso avanzados de flatMap

```javascript
// 1. Filtrar y transformar en una sola operación
// (devolver [] para "filtrar" un elemento)
const datos = [
  { nombre: 'Ana', activo: true, scores: [8, 9, 7] },
  { nombre: 'Bob', activo: false, scores: [6, 5, 9] },
  { nombre: 'Carlos', activo: true, scores: [10, 8, 9] }
];

// Obtener todos los scores de usuarios activos (flatMap combina filter + map + flat)
const scoresActivos = datos.flatMap(usuario =>
  usuario.activo ? usuario.scores : []  // [] para filtrar
);
// [8, 9, 7, 10, 8, 9]

// 2. Parsear y expandir texto
const textoConEtiquetas = [
  'precio: 10, cantidad: 5',
  'precio: 20, cantidad: 3',
];

const pares = textoConEtiquetas.flatMap(linea =>
  linea.split(', ').map(par => {
    const [clave, valor] = par.split(': ');
    return { clave, valor: Number(valor) };
  })
);
// [
//   { clave: 'precio', valor: 10 }, { clave: 'cantidad', valor: 5 },
//   { clave: 'precio', valor: 20 }, { clave: 'cantidad', valor: 3 }
// ]

// 3. Generar pares clave-valor desde un objeto
const precios = { manzana: 1.2, naranja: 0.8, plátano: 0.5 };

const entradas = Object.entries(precios).flatMap(([fruta, precio]) => [
  { tipo: 'fruta', valor: fruta },
  { tipo: 'precio', valor: precio }
]);

// 4. Aplanar respuestas paginadas
const paginas = [
  { datos: [1, 2, 3], pagina: 1 },
  { datos: [4, 5, 6], pagina: 2 },
  { datos: [7, 8, 9], pagina: 3 }
];
const todosDatos = paginas.flatMap(p => p.datos);  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

## Comparativa de rendimiento y legibilidad

```javascript
const matriz = [[1, 2], [3, 4], [5, 6]];

// Antes de flat/flatMap (ES5):
const aplanado1 = [].concat.apply([], matriz);
const aplanado2 = Array.prototype.concat.apply([], matriz);
const aplanado3 = matriz.reduce((acc, arr) => acc.concat(arr), []);

// Con flat (ES2019):
const aplanado4 = matriz.flat();  // ✅ Más legible

// Con flatMap para transformar:
const doble = matriz.flatMap(subArr => subArr.map(n => n * 2));
// [2, 4, 6, 8, 10, 12]
```

## Limitaciones a tener en cuenta

```javascript
// flat/flatMap solo aplanan UN nivel (flatMap equivale a flat(1))
const tresNiveles = [[1, [2, [3]]]];
tresNiveles.flatMap(x => x);     // [1, [2, [3]]]  — solo un nivel
tresNiveles.flat(Infinity);       // [1, 2, 3]      — usa flat con Infinity

// flatMap no puede aplanar más de un nivel
// Para más profundidad, combina flat(n) con map:
const multinivel = [[1, [2]], [3, [4]]];
multinivel.flat(2);               // [1, 2, 3, 4]
```

**Consejo de entrevista**: `flatMap` es especialmente poderoso como combinación de filter + map: retornar `[]` descarta el elemento, retornar `[valor]` lo incluye sin aplanar, y retornar `[a, b]` expande un elemento en dos. Es más eficiente que `.map().flat()` porque crea menos arrays intermedios. La pregunta de entrevista clásica es: "¿Cómo obtendrías todos los nombres de una lista de equipos donde cada equipo tiene un array de miembros?" — la respuesta elegante es `equipos.flatMap(equipo => equipo.miembros)`.
