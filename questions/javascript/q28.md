---
title: ¿Qué es la asignación por destructuring en JavaScript?
difficulty: medium

---

La destructuración permite extraer valores de arrays o propiedades de objetos y asignarlos a variables de forma declarativa.

## Ejemplo con arrays
```js
const coordenadas = [10, 20, 30];
const [x, y, z] = coordenadas;

console.log(x, y, z);
```

## Ejemplo con objetos
```js
const usuario = {
  nombre: 'Sara',
  rol: 'admin',
  activo: true,
};

const { nombre, rol } = usuario;
console.log(nombre, rol);
```

## Por qué importa
- hace el código más expresivo cuando quieres “desempaquetar” datos;
- permite valores por defecto;
- ayuda mucho al trabajar con parámetros, respuestas de API y arrays de retorno.

## Ejemplo con valores por defecto y renombrado
```js
const { nombre: alias, idioma = 'es' } = { nombre: 'Leo' };
console.log(alias);  // 'Leo'
console.log(idioma); // 'es'
```

## Para interiorizarlo
La destructuración no es solo azúcar sintáctico: te ayuda a mostrar qué partes del dato te importan realmente.
