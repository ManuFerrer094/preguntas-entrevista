---
title: ¿Qué es el método reduce() y cuándo usarlo?
difficulty: medium
tags: [javascript, array, reduce, functional-programming]
---

`Array.prototype.reduce()` es un método que recorre un array y acumula un resultado aplicando una función reductora a cada elemento. Es uno de los métodos más versátiles de JavaScript: puede reemplazar a `map`, `filter`, y combinarlos en una sola pasada.

## Sintaxis

```javascript
array.reduce(callback(acumulador, valorActual, índice, array), valorInicial);
```

## Uso básico: suma y producto

```javascript
const numbers = [1, 2, 3, 4, 5];

// Suma
const sum = numbers.reduce((acc, n) => acc + n, 0);
console.log(sum); // 15

// Producto
const product = numbers.reduce((acc, n) => acc * n, 1);
console.log(product); // 120

// Máximo sin Math.max
const max = numbers.reduce((acc, n) => n > acc ? n : acc, -Infinity);
console.log(max); // 5
```

## Reemplazar map y filter

```javascript
const users = [
  { name: 'Ana', age: 28, active: true },
  { name: 'Luis', age: 17, active: true },
  { name: 'Marta', age: 32, active: false },
  { name: 'Carlos', age: 25, active: true },
];

// filter + map en dos pasadas
const result1 = users
  .filter(u => u.active && u.age >= 18)
  .map(u => u.name);

// reduce en una sola pasada (más eficiente con arrays grandes)
const result2 = users.reduce((acc, user) => {
  if (user.active && user.age >= 18) {
    acc.push(user.name);
  }
  return acc;
}, []);

console.log(result2); // ['Ana', 'Carlos']
```

## Agrupar elementos por propiedad

```javascript
const orders = [
  { product: 'manzana', category: 'fruta', qty: 3 },
  { product: 'pera', category: 'fruta', qty: 2 },
  { product: 'leche', category: 'lácteo', qty: 5 },
  { product: 'yogur', category: 'lácteo', qty: 1 },
];

const groupedByCategory = orders.reduce((acc, order) => {
  const { category } = order;
  if (!acc[category]) acc[category] = [];
  acc[category].push(order);
  return acc;
}, {});

console.log(groupedByCategory);
// {
//   fruta: [{ product: 'manzana', ... }, { product: 'pera', ... }],
//   lácteo: [{ product: 'leche', ... }, { product: 'yogur', ... }]
// }
```

## Aplanar arrays (flatten)

```javascript
const nested = [[1, 2], [3, 4], [5, 6]];

const flat = nested.reduce((acc, arr) => acc.concat(arr), []);
console.log(flat); // [1, 2, 3, 4, 5, 6]

// Alternativa nativa (ES2019+)
const flat2 = nested.flat();
```

## Contar ocurrencias

```javascript
const fruits = ['manzana', 'pera', 'manzana', 'naranja', 'pera', 'manzana'];

const count = fruits.reduce((acc, fruit) => {
  acc[fruit] = (acc[fruit] || 0) + 1;
  return acc;
}, {});

console.log(count);
// { manzana: 3, pera: 2, naranja: 1 }
```

## Componer funciones con reduce

```javascript
// pipe: aplica funciones de izquierda a derecha
const pipe = (...fns) => (value) => fns.reduce((acc, fn) => fn(acc), value);

const processNumber = pipe(
  x => x * 2,
  x => x + 10,
  x => x.toString(),
  s => `Resultado: ${s}`
);

console.log(processNumber(5)); // 'Resultado: 20'
```

## Cuándo NO usar reduce

```javascript
// ❌ Para operaciones simples, map y filter son más legibles
const doubled = numbers.reduce((acc, n) => { acc.push(n * 2); return acc; }, []);
// ✅ Más claro con map:
const doubled2 = numbers.map(n => n * 2);

// ❌ reduce sin valor inicial puede fallar con arrays vacíos
[].reduce((acc, n) => acc + n); // TypeError: Reduce of empty array with no initial value

// ✅ Siempre incluir valor inicial
[].reduce((acc, n) => acc + n, 0); // 0
```

**Consejo de entrevista**: `reduce` es poderoso, pero la legibilidad importa. Úsalo cuando combinas operaciones (filter + map) o construyes estructuras de datos complejas (grupos, conteos, índices). Para casos simples, `map` y `filter` son más expresivos. Siempre proporciona el valor inicial para evitar errores con arrays vacíos.
