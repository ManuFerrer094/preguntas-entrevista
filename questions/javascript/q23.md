---
title: ¿Qué son las funciones de orden superior en JavaScript?
difficulty: medium

Una función de orden superior es una función que recibe otras funciones como argumento, devuelve una función o ambas cosas. Son una pieza central de JavaScript porque las funciones son valores de primera clase.

## Ejemplo de código
```js
const numeros = [1, 2, 3, 4];

const pares = numeros.filter((n) => n % 2 === 0);
const duplicados = numeros.map((n) => n * 2);
const suma = numeros.reduce((acc, n) => acc + n, 0);

console.log(pares);      // [2, 4]
console.log(duplicados); // [2, 4, 6, 8]
console.log(suma);       // 10
```

## Ejemplo propio
```js
function envolverConLog(fn) {
  return function (...args) {
    console.log('Llamando con', args);
    return fn(...args);
  };
}

const sumar = (a, b) => a + b;
const sumarConLog = envolverConLog(sumar);

console.log(sumarConLog(2, 3));
```

## Por qué importan
Permiten abstracción, reutilización y composición. En vez de repetir bucles y lógica ceremonial, expresas intención: transformar, filtrar, acumular, envolver, validar, diferir.

## Para interiorizarlo
Cada vez que veas una función que "modifica cómo se ejecuta otra función", probablemente estás delante de una función de orden superior. Entender eso te abre muchas puertas del lenguaje.
