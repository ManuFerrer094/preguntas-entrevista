title: ¿Qué son las arrow functions y en qué se diferencian de las funciones normales?

Las arrow functions son una sintaxis más compacta para escribir funciones, pero su diferencia importante no es visual: tienen reglas distintas para `this`, `arguments`, `super` y `new.target`.

## Ejemplo de código
```js
const sumar = (a, b) => a + b;
console.log(sumar(2, 3)); // 5
```

## Diferencia clave con `this`
```js
const persona = {
  nombre: 'Clara',
  saludarConFuncion() {
    return function () {
      return this.nombre;
    };
  },
  saludarConArrow() {
    return () => this.nombre;
  },
};

console.log(persona.saludarConArrow()()); // 'Clara'
```

## Qué conviene recordar
- No tienen su propio `this`; capturan el del contexto exterior.
- No sirven como constructores con `new`.
- No tienen su propio objeto `arguments`.
- Son muy útiles en callbacks y composición funcional.

## Para interiorizarlo
Usa arrow functions cuando quieras heredar el contexto exterior. Si necesitas una función con su propio `this`, probablemente quieres una función normal.
