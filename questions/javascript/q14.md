---
title: ¿Qué hace la sentencia return en JavaScript?
difficulty: easy
tags: [functions, control-flow, language-features]

---

`return` indica qué valor sale de una función y, además, detiene la ejecución de esa función en ese punto.

## Ejemplo de código
```js
function esPar(numero) {
  if (numero % 2 === 0) {
    return true;
  }

  return false;
}

console.log(esPar(4)); // true
```

## Matices importantes
- Si una función no hace `return` explícito, devuelve `undefined`.
- `return` corta la ejecución: lo que venga después no se ejecuta.
- Con salto de línea justo después de `return`, el punto y coma automático puede darte una sorpresa.

## Ejemplo de trampa común
```js
function mala() {
  return
  {
    ok: true
  };
}

console.log(mala()); // undefined
```

## Para interiorizarlo
Piensa en `return` como el contrato de salida de la función. Si una función es difícil de entender, revisa primero qué caminos de retorno tiene.
