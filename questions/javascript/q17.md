`NaN` significa "Not-a-Number", pero su nombre confunde un poco: sigue siendo un valor del tipo `number`. Representa el resultado de una operación numérica inválida o imposible de representar como número normal.

## Ejemplo de código
```js
console.log(Number('hola')); // NaN
console.log(Math.sqrt(-1)); // NaN
console.log(typeof NaN); // 'number'
```

## Matices importantes
- `NaN` es el único valor que no es igual a sí mismo.
- Para comprobarlo, usa `Number.isNaN(valor)` mejor que la global `isNaN`, que convierte tipos.

## Ejemplo de comprobación
```js
const valor = Number('hola');

console.log(valor === NaN); // false
console.log(Number.isNaN(valor)); // true
```

## Para interiorizarlo
Cuando un cálculo empieza a devolver `NaN`, no te quedes en el síntoma. Busca el punto exacto donde una operación esperaba un número válido y recibió otra cosa.
