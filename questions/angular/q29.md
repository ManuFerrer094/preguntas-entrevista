---
title: ¿Para qué sirve la directiva *ngFor en Angular?
difficulty: easy
tags: [angular, ngFor, directives, template]
---

La directiva **`*ngFor`** es una directiva estructural que itera sobre una colección (array u objeto iterable) y renderiza un fragmento de plantilla HTML para cada elemento. Es el equivalente Angular de un bucle `for...of`.

## Uso básico

```typescript
@Component({
  standalone: true,
  imports: [CommonModule],
  template: `
    <ul>
      <li *ngFor="let fruta of frutas">{{ fruta }}</li>
    </ul>
  `
})
export class ListaComponent {
  frutas = ['Manzana', 'Pera', 'Naranja', 'Uva'];
}
```

## Variables locales disponibles

`*ngFor` expone variables locales muy útiles:

```html
<ul>
  <li *ngFor="let item of productos;
              let i = index;
              let primero = first;
              let ultimo = last;
              let par = even;
              let impar = odd;
              let cantidad = count">
    <span [class.primero]="primero" [class.ultimo]="ultimo">
      {{ i + 1 }}/{{ cantidad }} - {{ item.nombre }}
      <em *ngIf="par">(fila par)</em>
    </span>
  </li>
</ul>
```

| Variable  | Tipo      | Descripción                              |
|-----------|-----------|------------------------------------------|
| `index`   | `number`  | Índice actual (comienza en 0)            |
| `first`   | `boolean` | `true` si es el primer elemento          |
| `last`    | `boolean` | `true` si es el último elemento          |
| `even`    | `boolean` | `true` si el índice es par               |
| `odd`     | `boolean` | `true` si el índice es impar             |
| `count`   | `number`  | Total de elementos en la colección       |

## trackBy: optimización de rendimiento

Sin `trackBy`, cuando el array cambia, Angular **destruye y recrea todos los elementos del DOM**, incluso si la mayoría no han cambiado. Con `trackBy`, Angular puede identificar qué elementos son nuevos y cuáles no necesitan actualizarse.

```typescript
@Component({
  standalone: true,
  imports: [CommonModule],
  template: `
    <!-- ❌ Sin trackBy: re-renderiza todo en cada cambio -->
    <div *ngFor="let usuario of usuarios">{{ usuario.nombre }}</div>

    <!-- ✅ Con trackBy: solo actualiza los elementos que cambian -->
    <div *ngFor="let usuario of usuarios; trackBy: trackPorId">
      {{ usuario.nombre }}
    </div>
  `
})
export class UsuariosComponent {
  usuarios: Usuario[] = [];

  trackPorId(index: number, usuario: Usuario): number {
    return usuario.id; // Identificador único y estable
  }
}
```

## Iterar sobre objetos (con keyvalue pipe)

```html
<!-- Iterar sobre las propiedades de un objeto -->
<dl>
  <ng-container *ngFor="let entrada of configuracion | keyvalue">
    <dt>{{ entrada.key }}</dt>
    <dd>{{ entrada.value }}</dd>
  </ng-container>
</dl>
```

## Nueva sintaxis @for (Angular 17+)

Angular 17 introduce una nueva sintaxis de control de flujo que **requiere `track`** (equivalente a `trackBy`), haciendo obligatoria la optimización:

```html
@for (usuario of usuarios; track usuario.id) {
  <div>{{ usuario.nombre }}</div>
} @empty {
  <p>No hay usuarios disponibles.</p>
}
```

El bloque `@empty` es una característica nueva: se muestra cuando la colección está vacía, eliminando la necesidad de un `*ngIf` adicional.

## Ejemplo completo con objetos

```typescript
interface Producto {
  id: number;
  nombre: string;
  precio: number;
  disponible: boolean;
}

@Component({
  standalone: true,
  imports: [CommonModule, CurrencyPipe],
  template: `
    <div class="grid">
      @for (producto of productos; track producto.id; let idx = $index) {
        <div class="card" [class.agotado]="!producto.disponible">
          <span class="numero">{{ idx + 1 }}</span>
          <h3>{{ producto.nombre }}</h3>
          <p>{{ producto.precio | currency:'EUR' }}</p>
          @if (!producto.disponible) {
            <span class="badge">Agotado</span>
          }
        </div>
      } @empty {
        <p>No hay productos en el catálogo.</p>
      }
    </div>
  `
})
export class CatalogoComponent {
  productos: Producto[] = [
    { id: 1, nombre: 'Laptop', precio: 999, disponible: true },
    { id: 2, nombre: 'Teclado', precio: 79, disponible: false },
  ];
}
```
