Angular incluye un módulo de animaciones basado en la **Web Animations API** que permite crear animaciones complejas de forma declarativa, directamente en los componentes TypeScript. Las animaciones se definen en los metadatos del componente y están completamente integradas con el ciclo de vida.

## Configuración

```bash
# Las animaciones ya vienen incluidas, solo necesitas importar el módulo
# En una app standalone (Angular 17+):
```

```typescript
// main.ts
import { bootstrapApplication } from '@angular/platform-browser';
import { provideAnimations } from '@angular/platform-browser/animations';

bootstrapApplication(AppComponent, {
  providers: [
    provideAnimations() // ← Habilita las animaciones
    // O usar provideAnimationsAsync() para lazy loading
  ]
});
```

## API principal de animaciones

El paquete `@angular/animations` proporciona las siguientes funciones:

- **`trigger()`**: define una animación y le da un nombre.
- **`state()`**: define el estilo de un estado concreto.
- **`style()`**: define propiedades CSS inline.
- **`animate()`**: define la transición (duración, easing).
- **`transition()`**: define cuándo ocurre la animación (de qué estado a qué estado).
- **`keyframes()`**: define animaciones paso a paso.
- **`group()`**: ejecuta animaciones en paralelo.
- **`sequence()`**: ejecuta animaciones en serie.

## Ejemplo: animación de fade y slide

```typescript
import {
  Component, signal
} from '@angular/core';
import {
  trigger, state, style, animate, transition
} from '@angular/animations';

@Component({
  selector: 'app-panel',
  standalone: true,
  animations: [
    trigger('fadeSlide', [
      state('visible', style({
        opacity: 1,
        transform: 'translateY(0)'
      })),
      state('oculto', style({
        opacity: 0,
        transform: 'translateY(-20px)'
      })),
      transition('oculto => visible', animate('300ms ease-out')),
      transition('visible => oculto', animate('200ms ease-in'))
    ])
  ],
  template: `
    <button (click)="alternar()">Mostrar/Ocultar</button>
    <div [@fadeSlide]="estado()">
      <p>Contenido animado</p>
    </div>
  `
})
export class PanelComponent {
  estado = signal<'visible' | 'oculto'>('oculto');

  alternar(): void {
    this.estado.update(s => s === 'visible' ? 'oculto' : 'visible');
  }
}
```

## Animaciones de entrada y salida (:enter / :leave)

Los alias `:enter` y `:leave` son atajos para `void => *` y `* => void`, muy útiles con `*ngIf` / `@if`:

```typescript
animations: [
  trigger('aparecer', [
    transition(':enter', [
      style({ opacity: 0, transform: 'scale(0.8)' }),
      animate('250ms ease-out', style({ opacity: 1, transform: 'scale(1)' }))
    ]),
    transition(':leave', [
      animate('200ms ease-in', style({ opacity: 0, transform: 'scale(0.8)' }))
    ])
  ])
]
```

```html
<div @aparecer *ngIf="mostrar">¡Aparezco con animación!</div>
```

## Animaciones con keyframes

```typescript
trigger('rebotar', [
  transition(':enter', [
    animate('600ms', keyframes([
      style({ transform: 'translateY(-100%)', offset: 0 }),
      style({ transform: 'translateY(10px)',  offset: 0.7 }),
      style({ transform: 'translateY(-5px)',  offset: 0.9 }),
      style({ transform: 'translateY(0)',     offset: 1 })
    ]))
  ])
])
```

## Animaciones en listas con query y stagger

```typescript
trigger('listaAnimada', [
  transition(':enter', [
    query(':enter', [
      style({ opacity: 0, transform: 'translateX(-20px)' }),
      stagger(100, [
        animate('300ms ease-out',
          style({ opacity: 1, transform: 'translateX(0)' }))
      ])
    ], { optional: true })
  ])
])
```

## Deshabilitar animaciones en tests

```typescript
import { provideNoopAnimations } from '@angular/platform-browser/animations';

// En el TestBed de los tests:
TestBed.configureTestingModule({
  providers: [provideNoopAnimations()]
});
```

**Consejo de entrevista**: Para deshabilitar animaciones en dispositivos que prefieren reducir el movimiento, usa el media query `prefers-reduced-motion` en combinación con `AnimationBuilder` o simplemente aplica estilos globales en CSS. Menciona también `provideAnimationsAsync()` disponible desde Angular 17, que carga el motor de animaciones de forma lazy, mejorando el tiempo de carga inicial.
