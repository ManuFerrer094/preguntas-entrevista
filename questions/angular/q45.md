**Angular Material** es la librería oficial de componentes UI para Angular, basada en las especificaciones de **Material Design** de Google. Proporciona un conjunto de componentes accesibles, bien testeados y con un sistema de temas (theming) muy flexible.

## Instalación

```bash
ng add @angular/material
```

El CLI te preguntará por:
- El **tema preconfigurado** (Indigo/Pink, Deep Purple/Amber, Pink/Blue Grey, Purple/Green) o personalizado.
- Si configurar la **tipografía de Angular Material** globalmente.
- Si incluir las **animaciones de Angular** (recomendado: sí).

## Componentes principales

### Formularios y entrada

```typescript
import { MatInputModule } from '@angular/material/input';
import { MatButtonModule } from '@angular/material/button';
import { MatSelectModule } from '@angular/material/select';
import { MatCheckboxModule } from '@angular/material/checkbox';
import { MatDatepickerModule } from '@angular/material/datepicker';

@Component({
  standalone: true,
  imports: [
    MatInputModule, MatButtonModule, MatSelectModule,
    MatCheckboxModule, ReactiveFormsModule
  ],
  template: `
    <form [formGroup]="form" (ngSubmit)="enviar()">
      <mat-form-field appearance="outline">
        <mat-label>Nombre</mat-label>
        <input matInput formControlName="nombre" placeholder="Tu nombre">
        <mat-error *ngIf="form.get('nombre')?.hasError('required')">
          El nombre es requerido
        </mat-error>
      </mat-form-field>

      <mat-form-field appearance="outline">
        <mat-label>País</mat-label>
        <mat-select formControlName="pais">
          <mat-option value="es">España</mat-option>
          <mat-option value="mx">México</mat-option>
          <mat-option value="ar">Argentina</mat-option>
        </mat-select>
      </mat-form-field>

      <button mat-raised-button color="primary" type="submit">
        Enviar
      </button>
    </form>
  `
})
export class FormularioMaterialComponent {
  form = new FormGroup({
    nombre: new FormControl('', Validators.required),
    pais: new FormControl('')
  });
}
```

### Diálogos

```typescript
import { MatDialog } from '@angular/material/dialog';

@Injectable()
export class GestionProductosService {
  constructor(private dialog: MatDialog) {}

  abrirConfirmacion(): Observable<boolean> {
    const ref = this.dialog.open(ConfirmacionDialogComponent, {
      width: '400px',
      data: { mensaje: '¿Estás seguro de eliminar este elemento?' }
    });
    return ref.afterClosed();
  }
}
```

## Sistema de temas (Theming)

Angular Material usa **CSS Custom Properties** y SCSS para la personalización:

```scss
// styles.scss
@use '@angular/material' as mat;

$paleta-primaria: mat.define-palette(mat.$indigo-palette);
$paleta-acento: mat.define-palette(mat.$pink-palette, A200, A100, A400);

$tema: mat.define-light-theme((
  color: (
    primary: $paleta-primaria,
    accent: $paleta-acento,
  ),
  typography: mat.define-typography-config(),
  density: 0,
));

@include mat.all-component-themes($tema);
```

## Accesibilidad

Angular Material sigue las pautas **WCAG 2.1** y el estándar **WAI-ARIA**:
- Todos los componentes tienen roles ARIA apropiados.
- Soporte completo de navegación por teclado.
- Contraste de colores conforme a las pautas de accesibilidad.
- Compatible con lectores de pantalla.
