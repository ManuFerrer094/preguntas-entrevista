---
title: ¿Qué es el Angular CDK?
difficulty: medium
tags: [angular, cdk, component-dev-kit, accessibility]
---

El Angular CDK (Component Development Kit) es una colección de herramientas y primitivas que ayudan a construir componentes UI de alta calidad sin imponer un estilo visual específico. Es la base sobre la que se construye Angular Material, pero puede usarse independientemente en cualquier proyecto.

El CDK proporciona comportamientos comunes que son difíciles de implementar correctamente desde cero: drag & drop, virtual scrolling, overlay (tooltips, modales), gestión del foco, accesibilidad, y mucho más.

**Drag and Drop:**

```typescript
import { DragDropModule, CdkDragDrop, moveItemInArray, transferArrayItem } from '@angular/cdk/drag-drop';

@Component({
  imports: [DragDropModule],
  template: `
    <div cdkDropList (cdkDropListDropped)="drop($event)">
      <div *ngFor="let item of items" cdkDrag>
        {{ item }}
      </div>
    </div>
  `
})
export class KanbanComponent {
  items = ['Tarea 1', 'Tarea 2', 'Tarea 3'];

  drop(event: CdkDragDrop<string[]>) {
    moveItemInArray(this.items, event.previousIndex, event.currentIndex);
  }
}
```

**Virtual Scrolling** para listas de miles de elementos:

```typescript
import { ScrollingModule } from '@angular/cdk/scrolling';

@Component({
  imports: [ScrollingModule],
  template: `
    <cdk-virtual-scroll-viewport itemSize="50" style="height: 400px;">
      <div *cdkVirtualFor="let item of items" style="height: 50px;">
        {{ item.name }}
      </div>
    </cdk-virtual-scroll-viewport>
  `
})
export class VirtualListComponent {
  items = Array.from({ length: 100000 }, (_, i) => ({ name: `Item ${i}` }));
}
```

**Overlay** para crear tooltips, menús desplegables y modales personalizados:

```typescript
import { Overlay, OverlayRef } from '@angular/cdk/overlay';
import { ComponentPortal } from '@angular/cdk/portal';

@Injectable()
export class TooltipService {
  private overlay = inject(Overlay);
  private overlayRef?: OverlayRef;

  show(origin: ElementRef, message: string) {
    const positionStrategy = this.overlay
      .position()
      .flexibleConnectedTo(origin)
      .withPositions([{ originX: 'center', originY: 'bottom', overlayX: 'center', overlayY: 'top' }]);

    this.overlayRef = this.overlay.create({ positionStrategy });
    const portal = new ComponentPortal(TooltipComponent);
    const ref = this.overlayRef.attach(portal);
    ref.instance.message = message;
  }

  hide() {
    this.overlayRef?.dispose();
  }
}
```

**FocusTrap** para accesibilidad en modales:

```typescript
import { A11yModule, FocusTrap, FocusTrapFactory } from '@angular/cdk/a11y';

// En un modal, atrapa el foco dentro del elemento
export class ModalComponent implements AfterViewInit {
  private el = inject(ElementRef);
  private focusTrapFactory = inject(FocusTrapFactory);
  private focusTrap!: FocusTrap;

  ngAfterViewInit() {
    this.focusTrap = this.focusTrapFactory.create(this.el.nativeElement);
    this.focusTrap.focusInitialElementWhenReady();
  }

  ngOnDestroy() {
    this.focusTrap.destroy();
  }
}
```

Otros módulos del CDK incluyen: **@angular/cdk/clipboard** (copiar al portapapeles), **@angular/cdk/stepper** (asistentes paso a paso), **@angular/cdk/table** (tablas de datos), y **@angular/cdk/layout** (breakpoints responsivos con `BreakpointObserver`).

**Consejo de entrevista**: El CDK es especialmente valioso cuando se necesita construir componentes accesibles y personalizados sin depender de Angular Material. Mencionar que el CDK maneja detalles como gestión del foco, compatibilidad con lectores de pantalla y comportamientos cross-browser demuestra conocimiento profundo de Angular.
