---
title: ¿Cuál es la diferencia entre AngularJS y Angular?
difficulty: easy

---

AngularJS (Angular 1.x) y Angular (2+) son dos frameworks completamente distintos que comparten nombre pero poco más. Angular 2+ fue una **reescritura total** del framework original, lanzada en 2016.

## AngularJS (Angular 1.x)

- Basado en el patrón **MVC** (Model-View-Controller).
- Escrito en **JavaScript** puro.
- Usa **$scope** y **controladores** para gestionar el estado.
- El sistema de **two-way data binding** era potente pero causaba problemas de rendimiento en aplicaciones grandes (ciclos de digest).
- Las dependencias se inyectan como **strings**, lo que no es seguro en minificación.
- No tiene soporte oficial para **aplicaciones móviles**.
- No existe el concepto de componentes como unidad principal.

```javascript
// AngularJS - controlador clásico
angular.module('miApp', [])
  .controller('HeroController', function($scope, HeroService) {
    $scope.heroes = [];
    HeroService.getHeroes().then(function(data) {
      $scope.heroes = data;
    });
  });
```

## Angular (2+)

- Basado en una **arquitectura de componentes**.
- Escrito en **TypeScript**, con tipado fuerte.
- Sin `$scope` ni controladores: los **componentes** son la unidad principal.
- Sistema de **inyección de dependencias** robusto y jerárquico.
- Soporte nativo para **aplicaciones móviles** (Angular + Ionic, NativeScript).
- Mejor rendimiento con **change detection** optimizable (`OnPush`, Signals).
- **Compilación AOT** por defecto en producción.
- CLI oficial (`@angular/cli`) para scaffolding y build.

```typescript
// Angular moderno - componente con TypeScript
@Component({
  selector: 'app-hero',
  standalone: true,
  template: `<h1>{{ hero.name }}</h1>`
})
export class HeroComponent {
  @Input() hero!: Hero;
}
```

## Tabla comparativa

| Característica       | AngularJS (1.x)       | Angular (2+)          |
|----------------------|-----------------------|-----------------------|
| Lenguaje             | JavaScript            | TypeScript            |
| Arquitectura         | MVC                   | Basada en componentes |
| Rendimiento          | Bajo (digest cycles)  | Alto (Ivy + OnPush)   |
| Mobile               | No                    | Sí                    |
| CLI oficial          | No                    | Sí (`ng`)             |
| Estado del proyecto  | End of Life (2022)    | Activo y mantenido    |
| Curva de aprendizaje | Media                 | Media-Alta            |

> AngularJS llegó a su **fin de vida** en diciembre de 2021. Todos los proyectos nuevos deben usar Angular 2+.
