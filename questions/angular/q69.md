Angular Router emite una secuencia de eventos durante cada navegación. Suscribirse a estos eventos permite implementar funcionalidades como indicadores de carga, seguimiento de analíticas, gestión de títulos de página y más.

## Eventos del Router

Los eventos principales en orden de ejecución:

```
NavigationStart       ← Inicio de la navegación
RoutesRecognized      ← Ruta reconocida
GuardsCheckStart      ← Comienzan a ejecutarse los Guards
GuardsCheckEnd        ← Guards completados
ResolveStart          ← Comienzan los Resolvers
ResolveEnd            ← Resolvers completados
NavigationEnd         ← Navegación completada con éxito
NavigationCancel      ← Navegación cancelada (guard retornó false)
NavigationError       ← Error durante la navegación
```

## Implementación básica: indicador de carga

```typescript
@Component({
  selector: 'app-root',
  standalone: true,
  imports: [RouterOutlet, CommonModule, AsyncPipe],
  template: `
    @if (cargando$ | async) {
      <div class="overlay-carga">
        <div class="spinner"></div>
        <p>Cargando...</p>
      </div>
    }
    <app-navbar></app-navbar>
    <router-outlet></router-outlet>
  `
})
export class AppComponent implements OnInit, OnDestroy {
  cargando$ = new BehaviorSubject<boolean>(false);
  private destroy$ = new Subject<void>();

  constructor(private router: Router) {}

  ngOnInit(): void {
    this.router.events.pipe(
      takeUntil(this.destroy$)
    ).subscribe(event => {
      if (event instanceof NavigationStart) {
        this.cargando$.next(true);
      }
      if (
        event instanceof NavigationEnd ||
        event instanceof NavigationCancel ||
        event instanceof NavigationError
      ) {
        this.cargando$.next(false);
      }
    });
  }

  ngOnDestroy(): void {
    this.destroy$.next();
    this.destroy$.complete();
  }
}
```

## Filtrar eventos específicos con filter

```typescript
import { Router, NavigationEnd } from '@angular/router';
import { filter, map } from 'rxjs/operators';

@Injectable({ providedIn: 'root' })
export class AnalyticaService {
  constructor(private router: Router) {
    // Solo escuchar NavigationEnd
    this.router.events.pipe(
      filter((event): event is NavigationEnd => event instanceof NavigationEnd),
      map(event => event.urlAfterRedirects)
    ).subscribe(url => {
      console.log('Página visitada:', url);
      // this.googleAnalytics.trackPageView(url);
    });
  }
}
```

## Actualizar el título de la página dinámicamente

```typescript
@Injectable({ providedIn: 'root' })
export class GestorTitulosService {
  constructor(
    private router: Router,
    private activatedRoute: ActivatedRoute,
    private title: Title
  ) {
    this.router.events.pipe(
      filter(event => event instanceof NavigationEnd)
    ).subscribe(() => {
      const titulo = this.obtenerTituloRuta(this.activatedRoute);
      this.title.setTitle(titulo ? `${titulo} | Mi App` : 'Mi App');
    });
  }

  private obtenerTituloRuta(route: ActivatedRoute): string {
    let child = route.firstChild;
    while (child) {
      if (child.firstChild) {
        child = child.firstChild;
      } else if (child.snapshot.data['titulo']) {
        return child.snapshot.data['titulo'];
      } else {
        return '';
      }
    }
    return '';
  }
}
```

## ActivatedRoute — Ruta activa actual

```typescript
@Component({ /* ... */ })
export class DetalleComponent implements OnInit {
  constructor(
    private route: ActivatedRoute,
    private router: Router
  ) {}

  ngOnInit(): void {
    // Escuchar cambios en los parámetros de la ruta actual
    this.route.params.subscribe(params => {
      const id = params['id'];
      this.cargarDatos(id);
    });

    // Escuchar cambios en los query params
    this.route.queryParams.subscribe(params => {
      this.filtro = params['filtro'] ?? '';
      this.pagina = Number(params['pagina'] ?? 1);
    });

    // Datos estáticos de la ruta (data)
    this.route.data.subscribe(data => {
      this.titulo = data['titulo'];
    });
  }
}
```

## routerLinkActive — Clase activa en enlaces de navegación

```html
<nav>
  <!-- Añade clase 'activo' cuando la ruta coincide -->
  <a routerLink="/inicio" routerLinkActive="activo">Inicio</a>
  <a routerLink="/productos" routerLinkActive="activo">Productos</a>
  <a routerLink="/usuarios" routerLinkActive="activo">Usuarios</a>

  <!-- exacto: solo activo cuando la ruta es exactamente esa -->
  <a routerLink="/" routerLinkActive="activo" [routerLinkActiveOptions]="{ exact: true }">
    Inicio
  </a>

  <!-- Múltiples clases -->
  <a routerLink="/perfil" routerLinkActive="activo destacado">Mi perfil</a>
</nav>
```

## Verificar si se puede navegar antes de salir (CanDeactivate)

```typescript
export const guardFormulario: CanDeactivateFn<FormularioComponent> = (component) => {
  if (component.formulario.dirty) {
    return confirm('¿Deseas salir? Los cambios no guardados se perderán.');
  }
  return true;
};

// En la configuración de rutas
{ path: 'editar', component: EditarComponent, canDeactivate: [guardFormulario] }
```

**Consejo de entrevista**: El patrón más preguntado en entrevistas es el **indicador de carga durante la navegación**, que se implementa suscribiéndose a `router.events` y filtrando `NavigationStart` / `NavigationEnd`. Recuerda siempre desuscribirte de `router.events` en `ngOnDestroy` para evitar memory leaks. En Angular 14+, también existe el `Router.getCurrentNavigation()` para acceder al estado de la navegación actual, y los Title resolvers automáticos con `TitleStrategy`.
