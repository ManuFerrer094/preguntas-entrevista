Las **directivas** son clases que añaden comportamiento adicional a los elementos del DOM en Angular. Son una de las características más poderosas del framework y permiten extender el HTML con funcionalidad personalizada.

## Tipos de directivas

Angular tiene tres tipos principales de directivas:

1. **Componentes**: Técnicamente son directivas con una plantilla asociada. Son el tipo más común.
2. **Directivas estructurales**: Modifican la estructura del DOM (añaden o eliminan elementos).
3. **Directivas de atributo**: Cambian la apariencia o comportamiento de un elemento existente.

## Directivas estructurales integradas

Utilizan el prefijo `*` como azúcar sintáctico:

```html
<!-- *ngIf - muestra/oculta elementos -->
<p *ngIf="isLoggedIn">Bienvenido, usuario!</p>

<!-- *ngFor - itera sobre listas -->
<li *ngFor="let item of items; let i = index">
  {{ i + 1 }}. {{ item.name }}
</li>

<!-- *ngSwitch - selección múltiple -->
<div [ngSwitch]="status">
  <span *ngSwitchCase="'active'">Activo</span>
  <span *ngSwitchCase="'inactive'">Inactivo</span>
  <span *ngSwitchDefault>Desconocido</span>
</div>
```

## Directivas de atributo integradas

```html
<!-- NgClass - clases dinámicas -->
<div [ngClass]="{ 'activo': isActive, 'error': hasError }">Contenido</div>

<!-- NgStyle - estilos dinámicos -->
<p [ngStyle]="{ 'color': textColor, 'font-size': fontSize + 'px' }">Texto</p>
```

## Crear una directiva personalizada

```typescript
import { Directive, ElementRef, HostListener, Input } from '@angular/core';

@Directive({
  selector: '[appResaltar]',
  standalone: true
})
export class ResaltarDirective {
  @Input() colorResaltado = 'yellow';

  constructor(private el: ElementRef) {}

  @HostListener('mouseenter')
  onMouseEnter(): void {
    this.el.nativeElement.style.backgroundColor = this.colorResaltado;
  }

  @HostListener('mouseleave')
  onMouseLeave(): void {
    this.el.nativeElement.style.backgroundColor = '';
  }
}
```

Uso en la plantilla:

```html
<p appResaltar colorResaltado="lightblue">
  Pasa el ratón por encima para resaltar.
</p>
```

## Generar una directiva con CLI

```bash
ng generate directive resaltar
# o forma corta:
ng g d resaltar
```

Las directivas son ideales cuando necesitas reutilizar comportamiento DOM sin crear un componente completo, como tooltips, validaciones visuales, permisos de acceso o efectos interactivos.
