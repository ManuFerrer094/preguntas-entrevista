Los formularios reactivos de Angular se construyen mediante tres clases fundamentales que forman una jerarquía: `FormControl`, `FormGroup` y `FormArray`. Todas extienden de la clase abstracta `AbstractControl`.

## FormControl — Control individual

Representa un **único campo** del formulario. Gestiona el valor del campo, su estado (válido/inválido, pristine/dirty, touched/untouched) y sus validadores.

```typescript
import { FormControl, Validators } from '@angular/forms';

const nombre = new FormControl('Ana García', [
  Validators.required,
  Validators.minLength(3)
]);

console.log(nombre.value);    // 'Ana García'
console.log(nombre.valid);    // true
console.log(nombre.touched);  // false

nombre.setValue('María');     // Cambia el valor
nombre.reset();               // Vuelve al valor inicial y limpia estado

// Suscribirse a cambios de valor
nombre.valueChanges.subscribe(valor => console.log('Nuevo valor:', valor));
nombre.statusChanges.subscribe(estado => console.log('Estado:', estado));
```

## FormGroup — Grupo de controles

Agrupa múltiples `FormControl` (u otros `FormGroup`/`FormArray`) en un **objeto**. El estado del grupo es inválido si cualquiera de sus controles es inválido.

```typescript
import { FormGroup, FormControl, Validators } from '@angular/forms';

const formularioRegistro = new FormGroup({
  nombre: new FormControl('', [Validators.required, Validators.minLength(2)]),
  email: new FormControl('', [Validators.required, Validators.email]),
  contrasena: new FormControl('', [Validators.required, Validators.minLength(8)]),
  aceptaTerminos: new FormControl(false, Validators.requiredTrue)
});

// Acceder a controles individuales
const emailControl = formularioRegistro.get('email');
const nombreControl = formularioRegistro.controls['nombre']; // O así

// Obtener el valor de todo el formulario
console.log(formularioRegistro.value);
// { nombre: '', email: '', contrasena: '', aceptaTerminos: false }

// Establecer valores
formularioRegistro.patchValue({ nombre: 'Ana', email: 'ana@test.com' }); // Parcial
formularioRegistro.setValue({                                             // Completo
  nombre: 'Ana',
  email: 'ana@test.com',
  contrasena: 'MiPass123!',
  aceptaTerminos: true
});
```

## FormArray — Array dinámico de controles

Gestiona una **lista dinámica** de controles. Ideal para formularios donde el usuario puede añadir o eliminar filas (habilidades, teléfonos, productos en un pedido).

```typescript
import { FormArray, FormControl, FormGroup, Validators } from '@angular/forms';

const pedidoForm = new FormGroup({
  cliente: new FormControl('', Validators.required),
  productos: new FormArray([
    new FormGroup({
      nombre: new FormControl('', Validators.required),
      cantidad: new FormControl(1, [Validators.required, Validators.min(1)]),
      precio: new FormControl(0, Validators.min(0))
    })
  ])
});

// Acceder al FormArray
const productosArray = pedidoForm.get('productos') as FormArray;
// O mejor tipado:
const productosArray2 = pedidoForm.controls['productos'] as FormArray<FormGroup>;

// Añadir un nuevo control
productosArray.push(new FormGroup({
  nombre: new FormControl(''),
  cantidad: new FormControl(1),
  precio: new FormControl(0)
}));

// Eliminar un control por índice
productosArray.removeAt(1);

// Número de controles
console.log(productosArray.length);
```

## Ejemplo completo con plantilla

```typescript
@Component({
  selector: 'app-form-pedido',
  standalone: true,
  imports: [ReactiveFormsModule, CommonModule],
  template: `
    <form [formGroup]="pedidoForm" (ngSubmit)="onSubmit()">
      <!-- Campo simple -->
      <input formControlName="cliente" placeholder="Cliente">

      <!-- FormArray de productos -->
      <div formArrayName="productos">
        @for (control of productosArray.controls; track $index; let i = $index) {
          <div [formGroupName]="i" class="fila-producto">
            <input formControlName="nombre" placeholder="Producto">
            <input formControlName="cantidad" type="number" min="1">
            <input formControlName="precio" type="number" min="0">
            <button type="button" (click)="eliminarProducto(i)">✕</button>
          </div>
        }
      </div>

      <button type="button" (click)="agregarProducto()">+ Añadir producto</button>

      <p>Total: {{ calcularTotal() | currency:'EUR' }}</p>

      <button type="submit" [disabled]="pedidoForm.invalid">Realizar pedido</button>
    </form>
  `
})
export class FormPedidoComponent {
  private fb = inject(FormBuilder);

  pedidoForm = this.fb.group({
    cliente: ['', Validators.required],
    productos: this.fb.array([this.crearProducto()])
  });

  get productosArray(): FormArray {
    return this.pedidoForm.get('productos') as FormArray;
  }

  crearProducto(): FormGroup {
    return this.fb.group({
      nombre: ['', Validators.required],
      cantidad: [1, [Validators.required, Validators.min(1)]],
      precio: [0, [Validators.required, Validators.min(0)]]
    });
  }

  agregarProducto(): void {
    this.productosArray.push(this.crearProducto());
  }

  eliminarProducto(index: number): void {
    this.productosArray.removeAt(index);
  }

  calcularTotal(): number {
    return this.productosArray.controls.reduce((total, control) => {
      const cantidad = control.get('cantidad')?.value ?? 0;
      const precio = control.get('precio')?.value ?? 0;
      return total + (cantidad * precio);
    }, 0);
  }

  onSubmit(): void {
    if (this.pedidoForm.valid) {
      console.log(this.pedidoForm.value);
    }
  }
}
```

## Resumen comparativo

| Clase         | Estructura       | Valor             | Caso de uso                         |
|---------------|------------------|-------------------|-------------------------------------|
| `FormControl` | Campo único      | Primitivo (string, number, boolean) | Input, select, checkbox |
| `FormGroup`   | Objeto de campos | Objeto `{key: value}` | Formulario o sección de formulario |
| `FormArray`   | Array de campos  | Array `[value1, value2]` | Listas dinámicas (añadir/quitar filas) |

**Consejo de entrevista**: Una distinción importante es entre `setValue()` y `patchValue()` en `FormGroup`: `setValue()` requiere que proporciones un valor para **todos** los controles del grupo (lanza error si falta alguno), mientras que `patchValue()` acepta un objeto parcial y solo actualiza los campos que se proporcionan. Para `FormArray`, no existe `patchValue()`; se manipula elemento por elemento con `push()`, `removeAt()`, `setControl()` y `reset()`.
