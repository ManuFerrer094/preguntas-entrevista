---
title: ¿Qué es el lazy loading en React con React.lazy?
difficulty: medium
tags: [react, lazy, code-splitting, performance, webpack]
---

`React.lazy` permite importar componentes de forma dinámica, dividiendo el bundle en chunks que se cargan bajo demanda. Combinado con `Suspense`, mejora el tiempo de carga inicial al no descargar código que el usuario no necesita inmediatamente.

## Sintaxis básica

```jsx
import { lazy, Suspense } from 'react';

// ✅ Importación dinámica — el chunk se carga cuando el componente se necesita
const Dashboard = lazy(() => import('./Dashboard'));

// ❌ Importación estática — siempre se incluye en el bundle principal
// import Dashboard from './Dashboard';

function App() {
  return (
    <Suspense fallback={<div>Cargando panel...</div>}>
      <Dashboard />
    </Suspense>
  );
}
```

## Code splitting por rutas (caso más común)

```jsx
import { lazy, Suspense } from 'react';
import { BrowserRouter, Routes, Route } from 'react-router-dom';

// Cada página es un chunk separado
const Home = lazy(() => import('./pages/Home'));
const Dashboard = lazy(() => import('./pages/Dashboard'));
const Settings = lazy(() => import('./pages/Settings'));
const AdminPanel = lazy(() => import('./pages/AdminPanel'));

function AppSkeleton() {
  return (
    <div className="page-skeleton">
      <div className="skeleton-header" />
      <div className="skeleton-content" />
    </div>
  );
}

function App() {
  return (
    <BrowserRouter>
      <nav>/* Navegación */</nav>
      <Suspense fallback={<AppSkeleton />}>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/dashboard" element={<Dashboard />} />
          <Route path="/settings" element={<Settings />} />
          <Route path="/admin" element={<AdminPanel />} />
        </Routes>
      </Suspense>
    </BrowserRouter>
  );
}
```

## Code splitting de componentes pesados

```jsx
// Componentes pesados que no son críticos para el primer render
const RichTextEditor = lazy(() => import('./RichTextEditor'));
const DataTable = lazy(() => import('./DataTable'));
const ChartComponent = lazy(() => import('./ChartComponent'));

function PostEditor({ post }) {
  const [isEditing, setIsEditing] = useState(false);

  return (
    <div>
      <button onClick={() => setIsEditing(true)}>Editar</button>

      {isEditing && (
        // RichTextEditor solo se carga cuando el usuario decide editar
        <Suspense fallback={<TextareaFallback />}>
          <RichTextEditor initialValue={post.content} />
        </Suspense>
      )}
    </div>
  );
}
```

## Preloading — cargar anticipadamente

```jsx
// Precargar el chunk antes de que el usuario lo necesite
const Dashboard = lazy(() => import('./Dashboard'));

function Home() {
  // Precargar Dashboard cuando el usuario hace hover sobre el botón
  function handleMouseEnter() {
    import('./Dashboard'); // Inicia la descarga aunque no renderice aún
  }

  return (
    <button
      onMouseEnter={handleMouseEnter}
      onClick={() => navigate('/dashboard')}
    >
      Ir al Dashboard
    </button>
  );
}
```

## Manejo de errores con ErrorBoundary

```jsx
import { lazy, Suspense, Component } from 'react';

class LazyErrorBoundary extends Component {
  state = { hasError: false };

  static getDerivedStateFromError() {
    return { hasError: true };
  }

  render() {
    if (this.state.hasError) {
      return (
        <div>
          <p>Error al cargar el componente.</p>
          <button onClick={() => this.setState({ hasError: false })}>
            Reintentar
          </button>
        </div>
      );
    }
    return this.props.children;
  }
}

const HeavyComponent = lazy(() => import('./HeavyComponent'));

function App() {
  return (
    <LazyErrorBoundary>
      <Suspense fallback={<Spinner />}>
        <HeavyComponent />
      </Suspense>
    </LazyErrorBoundary>
  );
}
```

## Named exports con lazy

```jsx
// React.lazy solo funciona con default exports
// Para named exports, crear un wrapper:

// ❌ No funciona directamente:
// const { UserForm } = lazy(() => import('./Forms'));

// ✅ Solución 1: re-exportar como default en un archivo wrapper
// UserFormLazy.js: export { UserForm as default } from './Forms';
const UserForm = lazy(() => import('./UserFormLazy'));

// ✅ Solución 2: inline wrapper
const UserForm = lazy(() =>
  import('./Forms').then(module => ({ default: module.UserForm }))
);
```

**Consejo de entrevista**: El code splitting por rutas es la optimización más impactante para aplicaciones grandes. Apunta a que el bundle inicial sea menor de 150KB para un primer cargue rápido. Combina `React.lazy` con `Suspense` y `ErrorBoundary` siempre. El preloading en hover/focus mejora la percepción de velocidad sin aumentar el bundle inicial.
