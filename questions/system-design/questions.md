# ¿Qué es el diseño de sistemas?

El diseño de sistemas es el proceso de definir la arquitectura, componentes e interfaces de un sistema.

---

# ¿Qué es la escalabilidad horizontal vs vertical?

**Vertical**: añadir más recursos a un servidor existente.
**Horizontal**: añadir más servidores al sistema.

---

# ¿Qué es un Load Balancer?

Un Load Balancer distribuye el tráfico entrante entre múltiples servidores.

---

# ¿Qué es el CAP Theorem?

El Teorema CAP establece que un sistema distribuido no puede garantizar simultáneamente Consistency, Availability y Partition Tolerance.

---

# ¿Qué es el patrón CQRS?

CQRS separa las operaciones de lectura (queries) de las de escritura (commands) para escalar independientemente.
