---
title: ¿Qué son los Generics en TypeScript?
difficulty: hard
tags: [typescript, generics]
---

Los Generics permiten crear componentes que pueden trabajar con múltiples tipos.

```typescript
function identity<T>(arg: T): T {
  return arg;
}

interface ApiResponse<T> {
  data: T;
  status: number;
}
```
