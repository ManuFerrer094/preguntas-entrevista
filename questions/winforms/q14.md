---
title: ¿Qué es Data Binding y cómo se usa BindingSource en WinForms?
difficulty: medium
tags: [winforms, data-binding, bindingsource, datagridview, enlace-datos]

**Data Binding** es el mecanismo que sincroniza automáticamente los datos de un origen (lista, objeto, base de datos) con los controles de la interfaz de usuario, evitando actualizar los controles manualmente.

**BindingSource** actúa como intermediario entre el origen de datos y los controles, facilitando la navegación, la edición y la actualización.

**Ejemplo básico — lista de objetos con BindingSource:**

```csharp
public class Cliente
{
    public int Id { get; set; }
    public string Nombre { get; set; }
    public string Email { get; set; }
}

public partial class ClientesForm : Form
{
    private BindingSource _bindingSource = new BindingSource();

    public ClientesForm()
    {
        InitializeComponent();
        CargarDatos();
    }

    private void CargarDatos()
    {
        var clientes = new List<Cliente>
        {
            new Cliente { Id = 1, Nombre = "Ana García", Email = "ana@mail.com" },
            new Cliente { Id = 2, Nombre = "Luis Pérez", Email = "luis@mail.com" }
        };

        _bindingSource.DataSource = clientes;

        // Enlazar el DataGridView
        dataGridView1.DataSource = _bindingSource;

        // Enlazar controles individuales
        txtNombre.DataBindings.Add("Text", _bindingSource, "Nombre");
        txtEmail.DataBindings.Add("Text", _bindingSource, "Email");
    }
}
```

**Navegación por los registros:**

```csharp
private void btnSiguiente_Click(object sender, EventArgs e)
    => _bindingSource.MoveNext();

private void btnAnterior_Click(object sender, EventArgs e)
    => _bindingSource.MovePrevious();

private void btnPrimero_Click(object sender, EventArgs e)
    => _bindingSource.MoveFirst();

private void btnUltimo_Click(object sender, EventArgs e)
    => _bindingSource.MoveLast();
```

**Actualizar datos al guardar:**

```csharp
private void btnGuardar_Click(object sender, EventArgs e)
{
    // Confirmar los cambios pendientes en el control activo
    _bindingSource.EndEdit();

    var clientes = _bindingSource.DataSource as List<Cliente>;
    GuardarEnBaseDeDatos(clientes);
}
```

**Ventajas de usar BindingSource:**
- Abstrae el tipo de origen de datos (listas, `DataTable`, `IQueryable`…).
- Permite filtrar y ordenar con las propiedades `Filter` y `Sort`.
- Notifica cambios bidireccionales cuando el origen implementa `INotifyPropertyChanged` o `IBindingList`.

> **Nota:** Para notificaciones automáticas al editar propiedades, implementa `INotifyPropertyChanged` en tu clase de modelo.
