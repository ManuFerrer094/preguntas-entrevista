---
title: ¿Qué es ClickOnce y cuándo se utiliza en WinForms?
difficulty: easy

---

**ClickOnce** es una tecnología de despliegue de Microsoft incluida en el .NET Framework que permite **publicar, instalar y actualizar aplicaciones Windows Forms** (y WPF) de forma sencilla, sin necesidad de privilegios de administrador en la mayoría de los casos.

**Características principales:**

- **Instalación simplificada:** El usuario instala la aplicación con un simple clic desde un navegador, una carpeta compartida o un servidor web.
- **Actualizaciones automáticas:** La aplicación puede verificar si hay nuevas versiones al iniciarse y descargarlas automáticamente en segundo plano.
- **Aislamiento:** Cada versión se instala en una ubicación aislada, evitando conflictos con otras versiones o aplicaciones.
- **Sin privilegios de administrador:** Puede instalarse en el perfil del usuario (`AppData`), lo que facilita el despliegue en entornos corporativos restringidos.

**Cómo publicar con ClickOnce en Visual Studio:**
1. Click derecho en el proyecto → **Publicar**.
2. Seleccionar ubicación de publicación (carpeta, FTP, servidor web).
3. Configurar opciones de actualización (frecuencia de comprobación, URL de actualización).
4. Hacer clic en **Publicar**.

**Limitaciones:**
- Diseñado principalmente para aplicaciones de escritorio Windows; no aplica a aplicaciones web o de consola.
- Para actualizaciones complejas o configuraciones avanzadas de instalación, se prefieren herramientas como **WiX** o **Inno Setup**.

```csharp
// Verificar y aplicar actualizaciones programáticamente
if (ApplicationDeployment.IsNetworkDeployed)
{
    ApplicationDeployment deployment = ApplicationDeployment.CurrentDeployment;
    if (deployment.CheckForUpdate())
        deployment.Update();
}
```
