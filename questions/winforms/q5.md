---
title: ¿Cuándo usar MaskedTextBox en lugar de TextBox en WinForms?
difficulty: easy
tags: [winforms, controles, maskedtextbox, textbox, validacion]
---

El control **`MaskedTextBox`** se utiliza cuando se necesita que el usuario ingrese datos en un **formato específico y predefinido**. A diferencia del `TextBox` normal, que acepta cualquier entrada de texto, el `MaskedTextBox` rechaza automáticamente los caracteres que no coincidan con el patrón definido por su propiedad `Mask`.

**Casos de uso comunes:**
- Números de teléfono: `(999) 000-0000`
- Fechas: `00/00/0000`
- Códigos postales: `00000`
- Números de tarjeta de crédito: `0000-0000-0000-0000`
- Número de seguridad social o DNI con formato fijo

**Caracteres especiales en la máscara:**
| Carácter | Significado |
|----------|-------------|
| `0` | Dígito obligatorio (0-9) |
| `9` | Dígito opcional |
| `L` | Letra obligatoria |
| `A` | Letra o dígito obligatorio |
| `&` | Cualquier carácter obligatorio |

```csharp
// Configuración en código
maskedTextBox1.Mask = "(999) 000-0000"; // Teléfono
maskedTextBox2.Mask = "00/00/0000";     // Fecha

// Obtener solo los dígitos del valor ingresado
string telefono = new string(maskedTextBox1.Text.Where(char.IsDigit).ToArray());
```

**Cuándo usar `TextBox`:**
Cuando el formato de entrada es libre o variable, como campos de nombre, descripción o búsqueda, donde no hay un patrón fijo que imponer.
