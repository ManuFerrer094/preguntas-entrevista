---
title: ¿Cómo se asegura que un formulario WinForms aparezca centrado en la pantalla?
difficulty: easy
tags: [winforms, formulario, startposition, ui]
---

Para que un formulario WinForms aparezca centrado en la pantalla al abrirse, se debe configurar la propiedad **`StartPosition`** con el valor **`FormStartPosition.CenterScreen`**.

**Formas de configurarlo:**

**1. Desde el diseñador de Visual Studio:**
- Seleccionar el formulario.
- En la ventana de propiedades, buscar `StartPosition`.
- Seleccionar `CenterScreen`.

**2. Desde código (en el constructor o antes de llamar `Show`):**

```csharp
public Form1()
{
    InitializeComponent();
    this.StartPosition = FormStartPosition.CenterScreen;
}
```

**Valores disponibles de `FormStartPosition`:**

| Valor | Descripción |
|-------|-------------|
| `Manual` | Usa la posición definida por la propiedad `Location`. |
| `CenterScreen` | Centra el formulario en la pantalla. |
| `WindowsDefaultLocation` | Deja que Windows decida la posición. |
| `WindowsDefaultBounds` | Windows decide posición y tamaño. |
| `CenterParent` | Centra el formulario respecto a su formulario padre. |

**Nota:** Si se quiere centrar un formulario **hijo** respecto a su formulario padre (al abrirlo como diálogo), se debe usar `CenterParent`:

```csharp
using (var frmDetalle = new FormDetalle())
{
    frmDetalle.StartPosition = FormStartPosition.CenterParent;
    frmDetalle.ShowDialog(this);
}
```
