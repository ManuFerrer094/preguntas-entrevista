---
title: ¿Cómo se personaliza el dibujado de controles en WinForms mediante Owner Drawing?
difficulty: hard
tags: [winforms, owner-drawing, ownerdrawmode, graphics, personalizar]
---

El **Owner Drawing** (dibujado por el propietario) permite reemplazar el renderizado predeterminado de un control WinForms con código GDI+ propio, consiguiendo apariencias completamente personalizadas.

**Controles que soportan Owner Drawing:**

`ListBox`, `ComboBox`, `ListView`, `TreeView`, `TabControl`, `MenuItem`, `ToolStripItem`, entre otros.

**Ejemplo: ListBox con filas de colores alternativos:**

```csharp
listBox1.DrawMode = DrawMode.OwnerDrawFixed;
listBox1.ItemHeight = 24;

listBox1.DrawItem += (sender, e) =>
{
    if (e.Index < 0) return;

    // Color de fondo alternado
    var backColor = (e.Index % 2 == 0) ? Color.White : Color.AliceBlue;

    if ((e.State & DrawItemState.Selected) == DrawItemState.Selected)
        backColor = SystemColors.Highlight;

    e.Graphics.FillRectangle(new SolidBrush(backColor), e.Bounds);

    var texto = listBox1.Items[e.Index].ToString();
    var foreColor = (e.State & DrawItemState.Selected) == DrawItemState.Selected
        ? SystemColors.HighlightText : Color.Black;

    using var font = new Font("Segoe UI", 10);
    e.Graphics.DrawString(texto, font, new SolidBrush(foreColor),
        e.Bounds.X + 4, e.Bounds.Y + 4);

    e.DrawFocusRectangle();
};
```

**Ejemplo: Panel con borde redondeado sobreescribiendo `OnPaint`:**

```csharp
public class RoundedPanel : Panel
{
    protected override void OnPaint(PaintEventArgs e)
    {
        var g = e.Graphics;
        g.SmoothingMode = System.Drawing.Drawing2D.SmoothingMode.AntiAlias;

        using var path = new System.Drawing.Drawing2D.GraphicsPath();
        path.AddRoundedRectangle(ClientRectangle, 12); // Requiere método de extensión propio sobre GraphicsPath
        g.FillPath(Brushes.White, path);
        g.DrawPath(Pens.Gray, path);
    }
}
```

**Consideraciones de rendimiento:**

- Evita crear objetos `Brush`, `Pen`, `Font` dentro del evento sin `using`; causan fugas de GDI handles.
- Llama a `e.Graphics.SetClip(e.Bounds)` para limitar el dibujado al área del ítem.
