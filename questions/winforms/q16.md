title: ¿Cómo harías un CRUD con DataGridView y binding de datos en WinForms?

**DataGridView** es el control tabular principal de WinForms. Permite mostrar, editar, añadir y eliminar filas de forma visual con soporte nativo de enlace de datos.

**Configuración básica con BindingSource:**

```csharp
public class Producto
{
    public int Id { get; set; }
    public string Nombre { get; set; }
    public decimal Precio { get; set; }
}

public partial class ProductosForm : Form
{
    private BindingSource _bs = new BindingSource();
    private List<Producto> _productos = new List<Producto>();

    public ProductosForm()
    {
        InitializeComponent();
        CargarDatos();
    }

    private void CargarDatos()
    {
        _productos = ObtenerProductos(); // Llamada a BD o servicio
        _bs.DataSource = _productos;
        dataGridView1.DataSource = _bs;
    }

    private List<Producto> ObtenerProductos() =>
        new List<Producto>
        {
            new Producto { Id = 1, Nombre = "Teclado",  Precio = 49.99m },
            new Producto { Id = 2, Nombre = "Ratón",    Precio = 29.99m }
        };
}
```

**Agregar un nuevo registro:**

```csharp
private void btnAgregar_Click(object sender, EventArgs e)
{
    var nuevo = new Producto
    {
        Id = _productos.Count + 1,
        Nombre = txtNombre.Text,
        Precio = decimal.Parse(txtPrecio.Text)
    };
    _productos.Add(nuevo);
    _bs.ResetBindings(false); // Fuerza la actualización del grid
}
```

**Editar el registro seleccionado:**

```csharp
private void btnEditar_Click(object sender, EventArgs e)
{
    if (_bs.Current is Producto seleccionado)
    {
        seleccionado.Nombre = txtNombre.Text;
        seleccionado.Precio = decimal.Parse(txtPrecio.Text);
        _bs.ResetCurrentItem(); // Refresca solo la fila actual
    }
}
```

**Eliminar el registro seleccionado:**

```csharp
private void btnEliminar_Click(object sender, EventArgs e)
{
    if (dataGridView1.CurrentRow != null)
    {
        _bs.RemoveCurrent();
    }
}
```

**Cargar datos del registro seleccionado en los campos de texto:**

```csharp
private void dataGridView1_SelectionChanged(object sender, EventArgs e)
{
    if (_bs.Current is Producto p)
    {
        txtNombre.Text = p.Nombre;
        txtPrecio.Text = p.Precio.ToString("F2");
    }
}
```

**Buenas prácticas con DataGridView:**

- Usa `ReadOnly = true` en el grid y controla los cambios con botones propios para mayor control.
- Para `BindingList<T>` en lugar de `List<T>`, las inserciones y eliminaciones actualizan el grid automáticamente sin necesidad de `ResetBindings`.
- Desactiva la adición automática de filas con `dataGridView1.AllowUserToAddRows = false` si gestionas la inserción desde un formulario dedicado.
- Aplica formato a columnas monetarias: `dataGridView1.Columns["Precio"].DefaultCellStyle.Format = "C2";`
