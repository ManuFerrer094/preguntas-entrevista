Vue Router es la librería oficial de enrutamiento para Vue.js. Permite crear aplicaciones de página única (SPA) con múltiples vistas, navegación con historial del navegador, rutas anidadas, navegación programática y guards de navegación.

**Configuración con Vue Router 4:**

```javascript
// router/index.ts
import { createRouter, createWebHistory } from 'vue-router';
import { useAuthStore } from '@/stores/auth';

const router = createRouter({
  history: createWebHistory(),
  routes: [
    {
      path: '/',
      component: () => import('@/layouts/MainLayout.vue'),
      children: [
        { path: '', name: 'home', component: () => import('@/pages/HomePage.vue') },
        {
          path: 'users',
          children: [
            { path: '', name: 'users', component: () => import('@/pages/UsersPage.vue') },
            { path: ':id', name: 'user-detail', component: () => import('@/pages/UserDetailPage.vue') }
          ]
        }
      ]
    },
    {
      path: '/login',
      name: 'login',
      component: () => import('@/pages/LoginPage.vue'),
      meta: { requiresGuest: true }
    },
    {
      path: '/admin',
      component: () => import('@/layouts/AdminLayout.vue'),
      meta: { requiresAuth: true, role: 'admin' },
      children: [
        { path: '', name: 'admin', component: () => import('@/pages/AdminDashboard.vue') }
      ]
    },
    { path: '/:pathMatch(.*)*', name: 'not-found', component: () => import('@/pages/NotFound.vue') }
  ]
});

// Guard global de navegación
router.beforeEach((to, from) => {
  const authStore = useAuthStore();

  if (to.meta.requiresAuth && !authStore.isLoggedIn) {
    return { name: 'login', query: { redirect: to.fullPath } };
  }

  if (to.meta.requiresGuest && authStore.isLoggedIn) {
    return { name: 'home' };
  }
});

export default router;
```

**Componentes de navegación:**

```vue
<template>
  <nav>
    <!-- router-link activa automáticamente la clase "router-link-active" -->
    <RouterLink to="/">Inicio</RouterLink>
    <RouterLink :to="{ name: 'users' }">Usuarios</RouterLink>
    <RouterLink
      :to="{ name: 'user-detail', params: { id: userId } }"
      active-class="text-blue-500"
    >
      Mi Perfil
    </RouterLink>
  </nav>

  <!-- Aquí se renderiza el componente de la ruta actual -->
  <RouterView />
</template>
```

**Composables de Vue Router:**

```vue
<script setup>
import { useRouter, useRoute } from 'vue-router';

const router = useRouter();
const route = useRoute();

// route.params, route.query, route.meta, route.name
console.log(route.params.id); // parámetro de la URL
console.log(route.query.search); // ?search=...

// Navegación programática
const goToUser = (id) => router.push({ name: 'user-detail', params: { id } });
const goBack = () => router.go(-1);
const replaceRoute = () => router.replace({ name: 'home' }); // sin añadir al historial

// Acceder a los datos meta de la ruta
const requiresAuth = route.meta.requiresAuth;
</script>
```

**Transitions entre rutas:**

```vue
<template>
  <RouterView v-slot="{ Component, route }">
    <Transition name="fade" mode="out-in">
      <component :is="Component" :key="route.path" />
    </Transition>
  </RouterView>
</template>

<style>
.fade-enter-active, .fade-leave-active { transition: opacity 0.2s; }
.fade-enter-from, .fade-leave-to { opacity: 0; }
</style>
```

**Scroll behavior:**

```javascript
const router = createRouter({
  scrollBehavior(to, from, savedPosition) {
    if (savedPosition) return savedPosition; // Restaurar posición al ir atrás
    if (to.hash) return { el: to.hash, behavior: 'smooth' }; // Anclas
    return { top: 0 }; // Scroll al inicio
  }
});
```

**Consejo de entrevista**: Los guards de navegación son clave para implementar autenticación y autorización. Hay tres tipos: globales (`beforeEach`), por ruta (`beforeEnter`) y en componente (`onBeforeRouteLeave`). El `meta` de las rutas permite añadir información personalizada accesible desde los guards.
