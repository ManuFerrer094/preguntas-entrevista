---
title: ¿Qué son las directivas en Vue?
difficulty: medium
tags: [vue, directives, template, custom-directives]
---

Las directivas en Vue son atributos especiales con el prefijo `v-` que aplican comportamientos reactivos al DOM. Vue incluye varias directivas integradas y también permite crear directivas personalizadas para encapsular lógica de manipulación del DOM.

## Directivas integradas principales

```vue
<template>
  <!-- v-if / v-else-if / v-else: renderizado condicional -->
  <div v-if="isLoggedIn">Bienvenido, {{ user.name }}</div>
  <div v-else-if="isLoading">Cargando...</div>
  <div v-else>Por favor, inicia sesión</div>

  <!-- v-show: visibilidad con CSS (el elemento siempre se renderiza) -->
  <div v-show="isVisible">Este elemento usa display:none/block</div>

  <!-- v-for: renderizar listas -->
  <ul>
    <li v-for="(item, index) in items" :key="item.id">
      {{ index + 1 }}. {{ item.name }}
    </li>
  </ul>

  <!-- v-bind (:): enlace dinámico de atributos -->
  <img :src="imageUrl" :alt="imageAlt" :class="{ active: isActive }" />
  <button :disabled="isLoading" :style="{ color: primaryColor }">Enviar</button>

  <!-- v-on (@): escuchar eventos -->
  <button @click="handleClick">Click</button>
  <input @input="onInput" @keyup.enter="onEnter" @blur.once="onBlur" />
  <form @submit.prevent="onSubmit">...</form>

  <!-- v-model: two-way binding -->
  <input v-model="username" />
  <textarea v-model="description" />
  <input type="checkbox" v-model="isChecked" />
  <select v-model="selectedOption">
    <option v-for="opt in options" :key="opt.value" :value="opt.value">
      {{ opt.label }}
    </option>
  </select>

  <!-- v-model con modificadores -->
  <input v-model.trim="name" />       <!-- Elimina espacios -->
  <input v-model.number="age" />      <!-- Convierte a número -->
  <input v-model.lazy="bio" />        <!-- Actualiza en blur en vez de input -->
</template>
```

## Directivas personalizadas

Las directivas personalizadas son ideales para encapsular lógica de manipulación del DOM que se reutiliza en múltiples lugares:

```javascript
// directivas/v-focus.ts
export const vFocus = {
  mounted(el) {
    el.focus();
  }
};

// directivas/v-click-outside.ts
export const vClickOutside = {
  mounted(el, binding) {
    el._clickOutside = (event) => {
      if (!el.contains(event.target)) {
        binding.value(event); // Ejecutar el callback
      }
    };
    document.addEventListener('click', el._clickOutside);
  },
  unmounted(el) {
    document.removeEventListener('click', el._clickOutside);
  }
};

// directivas/v-tooltip.ts
export const vTooltip = {
  mounted(el, binding) {
    const tooltip = document.createElement('div');
    tooltip.className = 'tooltip';
    tooltip.textContent = binding.value;
    el.appendChild(tooltip);
    el.style.position = 'relative';
  },
  updated(el, binding) {
    el.querySelector('.tooltip').textContent = binding.value;
  }
};
```

**Registro y uso:**

```javascript
// Registro global (main.ts)
import { vClickOutside } from './directives/v-click-outside';
app.directive('click-outside', vClickOutside);

// Registro local en componente (Composition API)
// (Solo declarar en el script y Vue la detecta automáticamente)
```

```vue
<script setup>
import { vFocus, vClickOutside } from '@/directives';

const closeMenu = () => { menuOpen.value = false; };
</script>

<template>
  <input v-focus />  <!-- Se enfoca automáticamente al montar -->

  <div v-click-outside="closeMenu" class="dropdown">
    <!-- El menú se cierra al hacer click fuera -->
  </div>

  <button v-tooltip="'Eliminar elemento'" @click="deleteItem">🗑️</button>
</template>
```

**Consejo de entrevista**: La diferencia entre `v-if` y `v-show` es importante: `v-if` destruye y recrea el elemento del DOM (costoso si cambia frecuentemente), mientras que `v-show` solo cambia `display: none` (el elemento siempre está en el DOM). Usar `v-show` para toggle frecuente y `v-if` cuando el contenido raramente cambia o tiene condiciones complejas.
