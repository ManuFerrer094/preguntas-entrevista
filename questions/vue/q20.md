title: ¿Qué son las transiciones y animaciones en Vue 3?
difficulty: medium

Vue 3 proporciona los componentes `<Transition>` y `<TransitionGroup>` para aplicar animaciones CSS o JavaScript cuando los elementos entran y salen del DOM, o cuando cambian de posición.

## Transition básica con CSS

```vue
<script setup>
import { ref } from 'vue';
const show = ref(true);
</script>

<template>
  <button @click="show = !show">Toggle</button>

  <!-- Transition envuelve un único elemento/componente -->
  <Transition name="fade">
    <p v-if="show">Texto animado</p>
  </Transition>
</template>

<style>
/* Vue añade estas clases automáticamente durante la transición */

/* Estado inicial de la entrada y estado final de la salida */
.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}

/* Durante la entrada: activa la transición */
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease;
}

/* Estado final de la entrada y estado inicial de la salida (el estado normal) */
.fade-enter-to,
.fade-leave-from {
  opacity: 1;
}
</style>
```

## Slide + fade combinados

```vue
<style>
.slide-fade-enter-active {
  transition: all 0.3s ease-out;
}
.slide-fade-leave-active {
  transition: all 0.2s cubic-bezier(1, 0.5, 0.8, 1);
}
.slide-fade-enter-from,
.slide-fade-leave-to {
  transform: translateX(20px);
  opacity: 0;
}
</style>

<template>
  <Transition name="slide-fade">
    <p v-if="show">Desliza y desvanece</p>
  </Transition>
</template>
```

## Transition con JavaScript hooks (más control)

```vue
<script setup>
import { ref } from 'vue';
import gsap from 'gsap'; // O cualquier librería de animación

const show = ref(true);

function onEnter(el, done) {
  gsap.fromTo(el,
    { opacity: 0, y: -30, scale: 0.8 },
    { opacity: 1, y: 0, scale: 1, duration: 0.4, ease: 'back.out', onComplete: done }
  );
}

function onLeave(el, done) {
  gsap.to(el, {
    opacity: 0, y: 30, scale: 0.8, duration: 0.3, onComplete: done
  });
}
</script>

<template>
  <!-- :css="false" desactiva las clases CSS automáticas -->
  <Transition
    :css="false"
    @enter="onEnter"
    @leave="onLeave"
    @before-enter="el => el.style.opacity = 0"
  >
    <div v-if="show" class="card">
      <h3>Tarjeta animada con GSAP</h3>
    </div>
  </Transition>
</template>
```

## TransitionGroup para listas

```vue
<script setup>
import { ref } from 'vue';

const items = ref([1, 2, 3, 4, 5]);

function addItem() {
  items.value.push(Math.max(...items.value) + 1);
}
function removeItem(item) {
  items.value = items.value.filter(i => i !== item);
}
function shuffle() {
  items.value = [...items.value].sort(() => Math.random() - 0.5);
}
</script>

<template>
  <button @click="addItem">Añadir</button>
  <button @click="shuffle">Ordenar aleatoriamente</button>

  <!-- TransitionGroup para listas — requiere key en cada elemento -->
  <TransitionGroup name="list" tag="ul">
    <li v-for="item in items" :key="item">
      {{ item }}
      <button @click="removeItem(item)">✕</button>
    </li>
  </TransitionGroup>
</template>

<style>
.list-enter-active,
.list-leave-active {
  transition: all 0.4s ease;
}

.list-enter-from {
  opacity: 0;
  transform: translateX(-30px);
}

.list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}

/* Animar el movimiento de elementos existentes (FLIP animation) */
.list-move {
  transition: transform 0.4s ease;
}

/* Necesario para que el FLIP funcione correctamente */
.list-leave-active {
  position: absolute;
}
</style>
```

## Transition entre componentes (modo out-in)

```vue
<script setup>
import { ref, shallowRef } from 'vue';
import TabA from './TabA.vue';
import TabB from './TabB.vue';

const currentTab = shallowRef(TabA);
</script>

<template>
  <button @click="currentTab = TabA">Tab A</button>
  <button @click="currentTab = TabB">Tab B</button>

  <!-- mode="out-in": el actual sale ANTES de que entre el nuevo -->
  <Transition name="fade" mode="out-in">
    <component :is="currentTab" :key="currentTab" />
  </Transition>
  <!-- mode="in-out": el nuevo entra ANTES de que salga el actual -->
</template>
```

## appear — animar en el primer render

```vue
<template>
  <!-- appear: aplica la animación de entrada también en el primer montado -->
  <Transition name="fade" appear>
    <div v-if="dataLoaded" class="content">
      Contenido cargado
    </div>
  </Transition>
</template>
```

**Consejo de entrevista**: Para animaciones simples, las clases CSS de Vue son suficientes. Para animaciones complejas o encadenadas, usa los JavaScript hooks con librerías como GSAP o anime.js y establece `:css="false"` para que Vue no interfiera. En `TransitionGroup`, el atributo `v-move` (con clase `name-move`) habilita las animaciones FLIP de posición. Siempre asegúrate de que `:key` sea estable y único en los elementos de una lista animada.
